RM25T0E _51 Engine / Hybrid System _029235 1HZ INTAKE / EXHAUST _0133248 EXHAUST PIPE A

1HZ INTAKE / EXHAUST  EXHAUST PIPE  INSTALLATION  


PROCEDURE

17410 1.INSTALL FRONT EXHAUST PIPE ASSEMBLY

a.

Temporarily install the No. 1 exhaust pipe support bracket to the front exhaust pipe assembly with the bolt.


b.

Install a new gasket and the front exhaust pipe assembly to the exhaust manifold with 3 new nuts.

Torque:
54 N*m (551 kgf*cm, 40 ft.*lbf)


c.

Install the No. 1 exhaust pipe support bracket to the cylinder block sub-assembly with the 2 bolts.

Torque:
24 N*m (245 kgf*cm, 18 ft.*lbf)


d.

Tighten the bolt.

Torque:
19.5 N*m (199 kgf*cm, 14 ft.*lbf)


17420 2.INSTALL CENTER EXHAUST PIPE ASSEMBLY

a.

Install the exhaust pipe damper to the center exhaust pipe assembly with 2 new bolts.

Torque:
21 N*m (214 kgf*cm, 15 ft.*lbf)


b.

Install the exhaust pipe support cushion to the center exhaust pipe assembly with the 2 nuts.

Torque:
5.5 N*m (56 kgf*cm, 49 in.*lbf)


c.

Install the exhaust pipe support cushion to the vehicle body with the 2 nuts.

Torque:
5.5 N*m (56 kgf*cm, 49 in.*lbf)


d.

Install a new gasket and the center exhaust pipe assembly to the front exhaust pipe assembly with 2 new bolts and 2 new nuts.

Torque:
48 N*m (489 kgf*cm, 35 ft.*lbf)


e.

for Double Cab:

Install a new gasket and the No. 2 center exhaust pipe assembly to the center exhaust pipe assembly with 2 new nuts.

Torque:
48 N*m (489 kgf*cm, 35 ft.*lbf)


17430 3.INSTALL TAILPIPE ASSEMBLY

a.

for Model Code #J71, #J76, Series:

Install the No. 2 exhaust pipe support stay and exhaust pipe clamp to the tailpipe assembly with the 2 nuts.

Torque:
10 N*m (102 kgf*cm, 7 ft.*lbf)


b.

for Model Code #J76, #J78, #J79 Series:

Install the exhaust pipe damper to the tailpipe assembly with 2 new bolts.

Torque:
21 N*m (214 kgf*cm, 15 ft.*lbf)


c.

Install the 2 exhaust pipe support cushions to the tailpipe assembly with the 4 nuts.

Torque:
5.5 N*m (56 kgf*cm, 49 in.*lbf)


d.

Install the 2 exhaust pipe support cushions to the vehicle body with the 4 nuts.

Torque:
5.5 N*m (56 kgf*cm, 49 in.*lbf)


e.

for Model Code #J78, #J79 Series:

Install the support bracket with the 2 nuts.

Torque:
5.5 N*m (56 kgf*cm, 49 in.*lbf)


f.

for Model Code #J71, #J76 Series:

Install a new gasket and the tailpipe assembly to the center exhaust pipe assembly with 2 new bolts and 2 new nuts.

Torque:
48 N*m (489 kgf*cm, 35 ft.*lbf)


g.

for Model Code #J78 Series:

Install a new gasket and the tailpipe assembly to the center exhaust pipe assembly with 3 new bolts and 3 new nuts.

Torque:
48 N*m (489 kgf*cm, 35 ft.*lbf)


h.

for Model Code #J79 Series:

i.

except Double Cab:

Install a new gasket and the tailpipe assembly to the center exhaust pipe assembly with 3 new bolts and 3 new nuts.

Torque:
48 N*m (489 kgf*cm, 35 ft.*lbf)


ii.

for Double Cab:

Install a new gasket and the tailpipe assembly to the No. 2 center exhaust pipe assembly with 3 new bolts and 3 new nuts.

Torque:
48 N*m (489 kgf*cm, 35 ft.*lbf)



4.INSPECT FOR EXHAUST GAS LEAK

a.

Inspect for an exhaust gas leak from the pipe connecting area.

If gas is leaking, tighten the areas necessary to stop the leak. Replace damaged parts as necessary.
