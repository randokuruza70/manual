4. INPUT INJECTOR COMPENSATION CODE(S) INTO ECM

NOTICE:
·

When a fuel injector is replaced, the new injector compensation code must be input into the ECM. When the ECM is replaced, all of the existing injector compensation codes must be input into the new ECM.

·

Injector compensation codes are unique, 30-digit, alphanumeric values printed on the head portion of each fuel injector. If an incorrect injector compensation code is input into the ECM, the engine assembly may rattle or engine idling may become rough. In addition, engine failure may occur and the life of the engine may be shortened.

·

When an injector compensation code is input into the ECM, the pilot quantity learning values stored in the ECM are initialized. Also, DTC P062F is stored when the pilot quantity learning values are initialized.

5. PILOT QUANTITY LEARNING

NOTICE:

Driving conditions during "Pilot Quantity Learning" using the GTS differ from normal as the EGR remains constantly ON. As a result, leaving the engine idling during "Pilot Quantity Learning" for long periods of time may damage the engine. Therefore, perform "Pilot Quantity Learning" immediately after the engine is warmed up (within approximately 20 minutes).


·

"Pilot Quantity Learning" is performed when fuel injectors have been replaced and injector compensation codes have been registered. "Pilot Quantity Learning(Detail)" is performed when there are engine problems but no DTCs are output and the technician wishes to clear the learned values and have the vehicle relearn the values.

·

This procedure uses the GTS to perform "Pilot Quantity Learning".

·

When replacing the fuel injector, engine or ECM, perform this procedure after performing injector compensation (manual ID code registration).

6. PILOT QUANTITY LEARNING (DETAIL)

NOTICE:

Driving conditions during "Pilot Quantity Learning" using the GTS differ from normal as the EGR remains constantly ON. As a result, leaving the engine idling during "Pilot Quantity Learning" for long periods of time may damage the engine. Therefore, perform "Pilot Quantity Learning" immediately after the engine is warmed up (within approximately 20 minutes).


·

"Pilot Quantity Learning" is performed when fuel injectors have been replaced and injector compensation codes have been registered. "Pilot Quantity Learning(Detail)" is performed when there are engine problems but no DTCs are output and the technician wishes to clear the learned values and have the vehicle relearn the values.

It takes time to clear the learned values and perform learning for all cylinders.

·

This procedure uses the GTS to perform "Pilot Quantity Learning(Detail)".

8. DESCRIPTION OF VIN

NOTICE:

The Vehicle Identification Number (VIN) or frame number must be input into a replacement ECM.


HINT:
