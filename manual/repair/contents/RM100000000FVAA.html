RM25T0E _58 Vehicle Interior _029469 HEATING / AIR CONDITIONING _0137863 REAR HEATER UNIT(for Quarter Panel) A

HEATING / AIR CONDITIONING  REAR HEATER UNIT(for Quarter Panel)  REASSEMBLY  


PROCEDURE

87104R 1.INSTALL REAR HEATER BLOWER MOTOR SUB-ASSEMBLY

a.

Install the rear heater blower motor sub-assembly with the 3 screws.


87105R 2.INSTALL REAR HEATER BLOWER FAN SUB-ASSEMBLY

a.

Install the washer.


b.

Install the rear heater blower fan sub-assembly with nut.

Torque:
2.8 N*m (29 kgf*cm, 25 in.*lbf)


87107R 3.INSTALL REAR HEATER UNIT SUB-ASSEMBLY

a.

Install the rear heater unit sub-assembly.


b.

Install the case with the 4 screws.


c.

Install the valve lever with the screw.


d.

Install the water unit valve assembly.


e.

Install the clamp clip with the screw.


f.

Rotate the snap to install the shaft as shown in the illustration.


g.

Install the air duct assembly with the 6 screws.


87138R 4.INSTALL REAR HEATER BLOWER RESISTOR

a.

Install the rear heater blower resistor with the 2 screws.


b.

Connect the connector and 2 clamps.


87255F 5.INSTALL HEATER UNIT REAR HEATER WATER INLET HOSE

a.

Connect the heater unit rear heater water inlet hose with the clip.

NOTICE:

Do not apply excessive force to the heater unit rear heater water inlet hose.



87256F 6.INSTALL HEATER UNIT REAR HEATER WATER OUTLET HOSE

a.

Connect the heater unit rear heater water outlet hose with the clip.

NOTICE:

Do not apply excessive force to the heater unit rear heater water outlet hose.



87212A 7.INSTALL REAR NO. 1 AIR DUCT

a.

Install the rear No. 1 air duct with the 6 screws.
