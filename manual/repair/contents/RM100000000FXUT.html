PROCEDURE

84290 1.INSTALL LIMIT SWITCH ASSEMBLY

a.

Install the limit switch assembly.


b.

Connect the limit switch assembly connector.


c.

Remove the 2 bolts and 2 brackets from the retaining holders and unlock the windshield frame.


d.

Raise the windshield frame and install the 2 bolts.


e.

Install the 2 bolts and 2 brackets to the windshield frame retaining holders on the hood.


f.

Raise the wiper arm.

HINT:

Use the same procedure for both wiper arms.



g.

Push the wiper arm backward and tighten the wiper arm retaining screw.

HINT:

Use the same procedure for both wiper arms.
