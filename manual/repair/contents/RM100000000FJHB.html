DIAGNOSIS SYSTEM


0.302,0.396 2.094,0.635 1.792,0.24 10 false *1
*1 Glow Indicator Light


Description

The emission control ECU contains a built-in self-diagnosis system which detects trouble with the engine signal network and then flashes the glow indicator light on the combination meter assembly.

The DTCs (Diagnostic Trouble Code) can be read by the number of blinks of the glow indicator light when the TC and CG terminals of the DLC3 are connected. When 2 or more codes are indicated, the lowest numbered code will be output first.

Glow Indicator Light Check Description

i.

The glow indicator light will turn on when the ignition switch is turned to ON and the engine is not running.

HINT:

If the glow indicator light does not turn on, proceed to troubleshooting for the combination meter assembly.



ii.

When the engine is started, the glow indicator light turns off. If the light remains on, the diagnosis system has detected a malfunction or abnormality in the system.
