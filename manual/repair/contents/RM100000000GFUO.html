THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  B2793  Transponder Chip Malfunction  B2794  Unmatched Encryption Code  B2795  Unmatched Key Code  


DESCRIPTION

DTC B2793 is stored when a malfunction is found in the key during key code registration or a key code is not registered normally. Replace the key if key code registration cannot be performed normally and this DTC is output.

DTC B2794 is stored when a key with an incomplete key code is inserted into the ignition key cylinder.

DTC B2795 is stored when a key with a key code that is not registered in the ECU is inserted into the ignition key cylinder.
