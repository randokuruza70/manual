PROCEDURE

52581A 1.INSTALL REAR BUMPER FILLER PROTECTOR

HINT:

Use the same procedure for all rear bumper filler protector.


a.

Install the rear bumper filler protector.


b.

Install a nose piece to an air riveter or hand riveter.


c.

Insert the mandrel part of a new rivet into the nose piece.


d.

Place the riveter upright against the rear bumper filler protector and install the rivet.

HINT:

If the rivet cannot be cut, pull it once and cut it.
