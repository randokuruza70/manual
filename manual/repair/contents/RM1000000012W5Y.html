BRAKE SYSTEM (OTHER)  BRAKE MASTER CYLINDER(w/ VSC)  REASSEMBLY  


PROCEDURE

04493 1.INSTALL BRAKE MASTER CYLINDER KIT

a.
0.823,0.99 0.854,1.135 false

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to the rubber parts as shown in the illustration.


b.

Install the No. 2 piston to the master cylinder body.

NOTICE:

Insert the piston straight so that the inside of the cylinder is not damaged.



c.

With the piston pushed in, insert the straight pin into the port of the reservoir installation area.

NOTICE:

Make sure that the straight pin and piston are securely installed.



d.

With the piston pushed in, install a new snap ring using snap ring pliers.


47299 2.INSTALL BRAKE MASTER CYLINDER RESERVOIR STRAINER

a.

Install the brake master cylinder reservoir strainer to the brake master cylinder reservoir assembly.


47230 3.INSTALL BRAKE MASTER CYLINDER RESERVOIR FILLER CAP ASSEMBLY

a.

Install the brake master cylinder reservoir filler cap assembly to the brake master cylinder reservoir assembly.

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to 2 new master cylinder reservoir grommets.


b.

Install the 2 master cylinder reservoir grommets to the brake master cylinder reservoir assembly.
