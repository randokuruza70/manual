BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  ANTI-LOCK BRAKE SYSTEM  FREEZE FRAME DATA  


FREEZE FRAME DATA


a.

Using the GTS, check the vehicle condition (ECU, sensor) when the brake system operates or a DTC is output.



CHECK FREEZE FRAME DATA WHEN BRAKE SYSTEM OPERATES


a.

Freeze frame data is stored when the brake system operates. The freeze frame data is updated each time the system operates.


b.

Once freeze frame data is stored when the system operates, it is not cleared until the DTC is cleared.


c.

The number of times (maximum: 31) the ignition switch has been turned from off to ON since the last time the system was activated can be stored and displayed.

HINT:
·

When the vehicle is stopped (vehicle speed below 7 km/h), turning the ignition switch from off to ON does not cause the value to increase.

·

The value "31" is displayed when the ignition switch was turned on more than 31 times.




d.

Check freeze frame data when the system operates

i.

Turn the ignition switch off.


ii.

Connect the GTS to the DLC3 and turn the ignition switch to ON.


iii.

Enter the following menus: Chassis / ABS/VSC/TRC / Utility.


iv.

Select "ABS History".

v.

Check the freeze frame data when the system operates.




CHECK FREEZE FRAME DATA WHEN DTC WAS STORED


a.

Freeze frame data is stored when a DTC is output.


b.

Once freeze frame data is stored when a DTC is output, it is not updated or cleared until the DTC is cleared.


c.

The number of times (maximum: 31) the ignition switch has been turned from off to ON since a DTC was output can be stored and displayed.

HINT:

The value "31" is displayed when the ignition switch was turned ON more than 31 times.



d.

Check freeze frame data when DTC is output

i.

Turn the ignition switch off.


ii.

Connect the GTS to the DLC3 and turn the ignition switch to ON.


iii.

Enter the following menus: Chassis / ABS/VSC/TRC / Trouble Codes.


iv.

Select a DTC to display the freeze frame data.


v.

Check the freeze frame data for the output DTC.
