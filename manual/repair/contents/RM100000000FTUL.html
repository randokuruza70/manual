68142 14.REMOVE REAR DOOR GLASS RUN

a.

w/o Power Window:

i.

Temporarily install the window regulator handle.


ii.

Move the rear door glass sub-assembly so that the door glass channel can be seen.


iii.

Remove the window regulator handle.



b.

w/ Power Window:

i.

Temporarily install the rear power window regulator switch assembly.


ii.

Connect the cable to the negative (-) battery terminal.


iii.

Move the rear door glass sub-assembly so that the door glass channel can be seen.


iv.

Disconnect the cable from the negative (-) battery terminal.


v.

Remove the rear power window regulator switch assembly.



c.

Remove the rear door glass run.
