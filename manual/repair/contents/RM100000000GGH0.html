SEAT BELT  REAR SEAT OUTER BELT ASSEMBLY(for Long Body)  INSTALLATION  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for RHD and LHD vehicles.

·

The procedure listed below is for LHD vehicles.

·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

73370 1.INSTALL REAR SEAT OUTER 3 POINT TYPE BELT ASSEMBLY LH

a.

Attach the 3 guides and temporarily install the rear seat outer 3 point type belt assembly LH with the bolt.


b.

Tighten the bolt.

Torque:
42 N*m (428 kgf*cm, 31 ft.*lbf)


c.

Connect the shoulder anchor of the rear seat outer 3 point type belt assembly LH with the bolt.

Torque:
42 N*m (428 kgf*cm, 31 ft.*lbf)


d.

Attach the 2 claws to close the rear seat outer 3 point type belt shoulder anchor cover.


e.

Connect the rear seat outer 3 point type belt floor anchor with the bolt.

Torque:
42 N*m (428 kgf*cm, 31 ft.*lbf)


f.

Check that the rear seat outer 3 point type belt is not twisted.
