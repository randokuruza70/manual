RM25T0E _58 Vehicle Interior _029924 DOOR LOCK _0138406 POWER DOOR LOCK CONTROL SYSTEM D

DOOR LOCK  POWER DOOR LOCK CONTROL SYSTEM  OPERATION CHECK  


CHECK ELECTRICAL DOOR LOCK OPERATION


a.

Check the basic function.

i.

Check that all doors lock when the door lock control switch (for manual operation) is turned to lock and all doors unlock when turned to unlock.


ii.

Using the key, check that all doors lock when the driver side door lock key cylinder is turned to lock and all doors unlock when turned to unlock.
