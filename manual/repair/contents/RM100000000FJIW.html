DATA LIST


HINT:

Using the GTS's Data List allows switch, sensor, actuator, and other item values to be read without removing any parts. Reading the Data List early in troubleshooting is one way to save time.


NOTICE:

In the table below, the values listed under "Normal Condition" are reference values. Do not depend solely on these reference values when deciding whether a part is faulty or not.

b.

Turn the ignition switch OFF.


c.

d.

Turn the ignition switch ON.


e.

Turn the GTS ON.


f.

Enter the following menus: Powertrain / Engine and ECT / Data List.

*: In this Active Test, the fuel of all cylinders is cut, and cranking occurs for approximately 10 seconds. At this time, the speed of each cylinder is measured. 8.

At first, the GTS's display will show each cylinder's engine rpm measurement to be extremely high. After approximately 10 seconds of engine cranking, each cylinder's engine rpm measurement will change to the actual engine rpm.
