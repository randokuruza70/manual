83320 1.INSTALL FUEL SENDER GAUGE ASSEMBLY

NOTICE:

If an fuel sender gauge assembly is removed from the fuel tank assembly, replace the fuel sender gauge assembly with a new one.


a.

Install a new fuel sender gauge assembly with the 5 screws.

Torque:
1.5 N*m (15 kgf*cm, 13 in.*lbf)


b.

Attach the wire harness clamp to the fuel tank assembly.


c.

for Model Code #J76 Series:

Attach the wire harness clamp to the frame.


d.

for Model Code #J76 Series:

Connect the fuel sender gauge connector.
