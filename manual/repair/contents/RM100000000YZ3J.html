NOTICE:
·

When an fuel injector is replaced, the new injector compensation code must be input into the ECM. When the ECM is replaced, all of the existing injector compensation codes must be input into the new ECM.

·

Injector compensation codes are unique, 30-digit, alphanumeric values printed on the head portion of each fuel injector. If an incorrect injector compensation code is input into the ECM, the engine assembly may rattle or engine idling may become rough. In addition, engine failure may occur and the life of the engine may be shortened.

·

When an fuel injector is replaced, perform pilot quantity learning. When the ECM is replaced, all of the existing pilot quantity learning values must be input into the new ECM.



HINT:

When replacing the ECM with a new one, if the injector compensation code, pilot quantity learning value and/or catalyst thermal deterioration value are not entered, DTC P062F will be stored.


When replacing the ECM with a new one, input all learning values and codes into the new ECM as follows:


a.

Prior to replacing the ECM, use the GTS to read and save each learning value and code from the original ECM.


b.

After installing a new ECM, input the saved learning values and codes into the new ECM using the GTS.


c.

Turn the ignition switch off and turn the GTS off.


d.

Wait for at least 30 seconds.


e.

Turn the ignition switch to ON and turn the GTS on.


f.

Clear DTC P062F stored in the ECM using the GTS.

HINT:
·

Each fuel injector has different fuel injection characteristics. In order to optimize the fuel injections, the ECM uses the injector compensation codes to balance the different fuel injections between each fuel injector.

·

When you first turn the ignition switch to ON after replacing the ECM and other component(s), DTC P062F is stored. Manually clear the DTC upon completion of injector compensation code registration.
