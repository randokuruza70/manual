BRAKE (REAR)  REAR BRAKE(for Rear Disc Brake)  REASSEMBLY  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

47547C 1.TEMPORARILY INSTALL REAR DISC BRAKE BLEEDER PLUG

a.

Temporarily install the rear disc brake bleeder plug to the rear disc brake cylinder LH.

HINT:

The rear disc brake bleeder plug will be tightened to the torque specification in the "Bleed Brake Line" procedure.



31478C 2.INSTALL REAR DISC BRAKE BLEEDER PLUG CAP

3.INSTALL PISTON SEAL

a.
0,0 0,0 false

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to the entire inner and outer circumference of a new piston seal.


b.

Install the piston seal to the rear disc brake cylinder assembly LH.

NOTICE:

Make sure the piston seal is securely installed to the groove of the rear disc brake cylinder assembly LH.



4.INSTALL CYLINDER BOOT

a.
0,0 0,0 false

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to the entire inner circumference of a new cylinder boot.


b.

Apply a light coat of lithium soap base glycol grease to the entire outer circumference of the rear disc brake piston where it contacts the rear disc brake cylinder LH.


c.

Install the cylinder boot to the rear disc brake piston.


d.

Install the rear disc brake piston to the rear disc brake cylinder LH.

NOTICE:

Do not forcibly install the piston into the disc brake cylinder LH.



5.INSTALL REAR DISC BRAKE PISTON

a.

Install the cylinder boot to the rear disc brake cylinder LH.

NOTICE:

Install the cylinder boot securely to the groove of the disc brake cylinder LH.



b.

Push in the rear disc brake piston by hand and install the cylinder boot into the groove of the rear disc brake piston.

NOTICE:

Install the cylinder boot securely to the groove of the rear disc brake piston.
