1VD-FTV ENGINE CONTROL  ACCELERATOR PEDAL  INSTALLATION  


PROCEDURE

78110K 1.INSTALL ACCELERATOR PEDAL SENSOR ASSEMBLY

NOTICE:
·

Avoid physical shocks to the accelerator pedal sensor assembly.

·

Do not disassemble the accelerator pedal sensor assembly.

·

This accelerator pedal sensor assembly does not require lubrication.

·

Do not apply oil or other lubrication to the accelerator pedal sensor assembly.

If applied, the accelerator pedal sensor assembly must be replaced.



a.

Connect the tip of the accelerator pedal sensor assembly to the bracket.


b.

Install the accelerator pedal sensor assembly with the 2 nuts.

Torque:
5.0 N*m (51 kgf*cm, 44 in.*lbf)


c.

Connect the connector to the accelerator pedal sensor assembly.
