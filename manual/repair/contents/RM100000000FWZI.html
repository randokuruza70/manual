47750 2.INSTALL DISC BRAKE CYLINDER ASSEMBLY LH

a.

Install the disc brake cylinder assembly LH with the 2 bolts.

Torque:
123 N*m (1254 kgf*cm, 91 ft.*lbf)


47313A 3.CONNECT FRONT FLEXIBLE HOSE LH

a.

Install the front flexible hose LH and a new gasket to the disc brake cylinder assembly LH with a new union bolt.

Torque:
30.4 N*m (310 kgf*cm, 22 ft.*lbf)

HINT:

Install the flexible hose lock securely in the lock hole in the disc brake cylinder assembly LH.

Disc brake grease


a.

Apply disc brake grease to both sides of each No. 1 anti-squeal shim.


b.

Install the 2 No. 1 anti-squeal shims and 2 No. 2 anti-squeal shims to each front disc brake pad.

NOTICE:
·

When replacing worn pads, the shims must be replaced together with the pads.

·

Install the shims in the correct position and direction.




04465 5.INSTALL FRONT DISC BRAKE PAD

a.

Install the 2 front disc brake pads to the disc brake cylinder assembly LH.

NOTICE:

There should be no oil or grease on the friction surface of the pads and disc.



6.INSTALL FRONT DISC BRAKE ANTI-RATTLE SPRING

a.

Install the front disc brake anti-rattle spring.


b.

Hold down the front disc brake anti-rattle spring and insert the 2 disc brake pad guide pins.


c.

Install the pin hold clip.
