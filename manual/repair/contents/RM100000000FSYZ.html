DOOR LOCK  REAR DOOR LOCK  INSTALLATION  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

69140 1.INSTALL REAR DOOR LOCK ACTUATOR ASSEMBLY LH (w/ Power Door Lock)

a.

Attach the guide in the direction indicated by the arrow in the illustration.


b.

Install the rear door lock actuator assembly LH to the rear door lock assembly LH with the 2 screws.


69340 2.INSTALL REAR DOOR LOCK ASSEMBLY LH

a.

Apply MP grease to the sliding parts of the rear door lock assembly LH.


b.

Insert the rear door lock assembly LH into the rear door outside handle release plate, and set it on the rear door panel.


c.

Check that the rear door outside handle release plate is securely connected to the rear door lock assembly.


d.

Using a T30 "TORX" socket wrench, install the rear door lock assembly LH with the 3 screws.

Torque:
5.0 N*m (51 kgf*cm, 4 ft.*lbf)


e.

Connect the connector.


f.

Install the bolt.

Torque:
8.0 N*m (82 kgf*cm, 71 in.*lbf)


g.

Attach the clamp.
