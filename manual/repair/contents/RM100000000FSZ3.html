69350 2.INSTALL BACK DOOR LOCK ASSEMBLY

a.

Apply MP grease to the sliding parts of the back door lock assembly.


b.

w/ Power Door Lock:

i.

Using a T30 "TORX" socket wrench, install the back door lock assembly with the 3 screws.

Torque:
5.0 N*m (51 kgf*cm, 44 in.*lbf)


ii.

Attach the 2 snaps and connect the 2 control links as shown in the illustration.


iii.

Connect the 2 connectors.


iv.

Attach the clamp.



c.

w/o Power Door Lock:

i.

Using a T30 "TORX" socket wrench, install the back door lock assembly with the 3 screws.

Torque:
5.0 N*m (51 kgf*cm, 44 in.*lbf)


ii.

Attach the 2 snaps and connect the 2 control links as shown in the illustration.
