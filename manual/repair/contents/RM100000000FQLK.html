PROCEDURE

1.INSPECT DIFFERENTIAL SIDE GEAR BACKLASH

a.

Install the 2 rear No. 1 differential side gear thrust washers to the 2 rear differential side gears.


b.

Install the 4 rear differential pinion thrust washers to the 4 rear differential pinions.


c.

Install the 2 rear differential side gears into the differential case.


d.

Install the rear differential pinion shaft holder into the differential case.


e.

Install the 4 rear differential pinions and 4 rear differential pinion thrust washers.


f.

Align the holes of the differential case and pinion shaft, and install the 2 rear No. 1 differential pinion shafts and rear differential pinion shaft.


g.

Install the rear differential side gear to the differential cover.


h.

Align the matchmarks and temporarily install the 5 bolts and 3 pinion shaft pins.


i.

Using a T10 "TORX" socket, tighten the 5 bolts and 3 pinion shaft pins.

Torque:
57.9 N*m (590 kgf*cm, 43 ft.*lbf)


j.

Using a dial indicator, while holding the rear differential side gear, measure the backlash.

Standard backlash:

0.02 to 0.15 mm (0.00079 to 0.00590 in.)


If the backlash is not within the specification, install a thrust washer of a different thickness.

HINT:
·

Make sure the washers on both sides are the same size.

·

Measure at all 4 locations.

·

Apply hypoid gear oil to each sliding surface and rotating part.



Standard Thrust Washers Thickness:
Thickness Thickness

k.

4.INSTALL DIFFERENTIAL RING GEAR

a.

b.

c.

d.

Carefully take the differential ring gear out of the boiling water.


e.

After the moisture on the differential ring gear has completely evaporated, quickly install the differential ring gear to the differential case.

h.

After the differential ring gear cools down sufficiently, tighten the set bolts.

137 N*m (1397 kgf*cm, 101 ft.*lbf)


5.INSPECT DIFFERENTIAL RING GEAR

a.

Place the bearing outer races on their respective bearings.

HINT:

Do not interchange the left and right outer races.



b.

Install the assembled plate washers onto the rear differential case bearing (outer race).


c.

Install the differential case onto the differential carrier.

HINT:

If it is difficult to install the differential case onto the differential carrier, replace the plate washer with a thinner one.

However, select a plate washer that allows no clearance between it and the differential carrier.



d.

Align the matchmarks on the bearing caps and differential carrier.


e.

Install and uniformly tighten the 4 bolts a little at a time.


f.

Using a dial indicator, check the ring gear runout.

Maximum runout:

0.05 mm (0.00197 in.)



g.

Remove the differential case.

41302 10.INSTALL REAR DIFFERENTIAL CASE SUB-ASSEMBLY

a.

Place the rear differential case bearings (outer race) on their respective bearings.

HINT:

Do not interchange the left and right outer races.



b.

Install the assembled plate washer onto the rear differential case bearing.


c.

Install the differential case onto the differential carrier.


d.

Using a plastic-faced hammer, tap on the differential ring gear to make sure the differential case is aligned with the differential carrier, and fit the rear differential case bearings and plate washers into the differential carrier.

HINT:

If it is difficult to install the differential case onto the differential carrier, replace the plate washer with a thinner one.

However, select a plate washer that allows no clearance between it and the differential carrier.
