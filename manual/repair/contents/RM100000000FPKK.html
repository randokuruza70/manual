1HZ STARTING  STARTER(for 2.5 kW, 4.5 kW Type)  REASSEMBLY  


PROCEDURE

28160 1.INSTALL STARTER ARMATURE ASSEMBLY

a.

Install the starter armature assembly to the starter yoke assembly.


28140 2.INSTALL STARTER BRUSH HOLDER ASSEMBLY

a.

Install the starter brush holder assembly.


b.

Connect the 4 brushes to the starter brush holder assembly.

i.

Using a screwdriver, hold back the spring.


ii.

Connect the brush to the starter brush holder assembly.

NOTICE:

Check that positive (+) lead wires are not grounded.




c.

Install the dust protector to the starter commutator end frame.


d.

Install the starter commutator end frame to the starter yoke assembly with the 2 screws.

Torque:
3.6 N*m (37 kgf*cm, 32 in.*lbf)


28150 3.INSTALL MAGNET STARTER SWITCH ASSEMBLY

a.

Apply high-temperature grease to the steel ball, return spring, idle gear, retainer and clutch roller.


b.

Insert the steel ball into the starter drive housing assembly.
