REAR SUSPENSION  REAR LEAF SPRING  INSTALLATION  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

48306 1.INSTALL REAR SPRING BUMPER SUB-ASSEMBLY

a.

Install the bumper with the 2 bolts.

Torque:
29 N*m (296 kgf*cm, 21 ft.*lbf)


48210Q 2.INSTALL REAR SPRING SHACKLE BUSH

a.

Install the 4 bushes to the spring and chassis frame.


48220 3.TEMPORARILY INSTALL REAR SPRING ASSEMBLY LH

a.

Temporarily install the spring to the rear side with the shackle pin, shackle plate, 2 washers and 2 nuts.


b.

Temporarily install the spring to the front side with the hanger pin, washer and nut.


48475R 4.INSTALL REAR DISTANCE PLATE (w/ Distance Plate)

a.

Install the distance plate.


48046A 5.TEMPORARILY INSTALL REAR SPRING U BOLT

a.

Temporarily install the U bolt seat and 2 U bolts with the 4 washers and 4 new nuts.

NOTICE:

Install the U bolts so that the lengths of the 4 ends under the U bolt seat are the same.



48540 6.TEMPORARILY INSTALL REAR SHOCK ABSORBER ASSEMBLY LH

a.

Temporarily install the lower mounting bolt, rear No. 1 shock absorber cushion washer and 2 bushing and rear shock absorber assembly LH to the spring U bolt seat sub-assembly LH.


7.INSTALL REAR WHEEL

8.STABILIZE SUSPENSION

a.

Remove the safety stands.


b.

Lower the vehicle.


c.

Bounce the vehicle up and down several times to stabilize the suspension.


48220 9.TIGHTEN REAR SPRING ASSEMBLY LH

a.

Tighten the 3 nuts.

Torque:
for front side : 65 N*m (663 kgf*cm, 48 ft.*lbf)
for rear side : 38 N*m (387 kgf*cm, 28 ft.*lbf)


b.

Install the lock bolt.

Torque:
24 N*m (245 kgf*cm, 18 ft.*lbf)


48046A 10.TIGHTEN REAR SPRING U BOLT

a.

Tighten the 4 nuts.

Torque:
80 N*m (816 kgf*cm, 59 ft.*lbf)

NOTICE:

Install the U bolts so that the lengths of the 4 ends under the U-bolt seat are the same.



48540 11.TIGHTEN REAR SHOCK ABSORBER ASSEMBLY LH

a.

Tighten the bolt.

Torque:
64 N*m (653 kgf*cm, 47 ft.*lbf)
