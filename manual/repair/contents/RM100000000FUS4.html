FRONT SUSPENSION  FRONT COIL SPRING  INSTALLATION  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

48331C 1.INSTALL FRONT SPRING BUMPER LH

a.

Install the front spring bumper LH to the frame with the 2 bolts.

Torque:
28.4 N*m (290 kgf*cm, 21 ft.*lbf)


48310A 2.INSTALL FRONT SUSPENSION HOLLOW SPRING ASSEMBLY LH

a.

Install the front suspension hollow spring assembly LH to the frame with the 2 nuts.

Torque:
9.2 N*m (94 kgf*cm, 81 in.*lbf)


48132 3.INSTALL FRONT COIL SPRING LH

a.

Using SST, compress the front coil spring LH.

SST
09727-30022  (09727-00022, 09727-00031)

NOTICE:

Do not use an impact wrench.



b.
2.271,1.01 2.823,1.01 true 2.875,0.948 3.229,1.135 0.354,0.188 10 true SST

Install the front coil spring LH to the front axle housing assembly.

HINT:

Before installing the front coil spring LH, check that the coil spring end is in the correct position. If not, reinstall the front coil spring LH.



c.

Jack up the front axle housing assembly.


d.

Remove SST from the front coil spring LH.

NOTICE:

Do not use an impact wrench.



48811 4.TEMPORARILY INSTALL FRONT STABILIZER BAR

a.

Temporarily install the front stabilizer bar to the front axle housing assembly with the bolt and nut.

a.

Lower the vehicle.

Tighten the bolt and nut of the front stabilizer bar.
