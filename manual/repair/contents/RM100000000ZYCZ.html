CUSTOMIZE METER / GAUGE SYSTEM


a.

Customizing with the GTS

i.

Connect the GTS to the DLC3.


ii.

Turn the ignition switch to ON.


iii.

Turn the GTS on.


iv.

Enter the following menus: Customize Setting / Warning or Display.


v.

Select the setting by referring to the table below.

NOTICE:

Be sure to record the current value before customizing.
