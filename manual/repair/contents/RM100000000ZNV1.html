BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  VEHICLE STABILITY CONTROL SYSTEM  ABS Warning Light Remains ON  


DESCRIPTION

The skid control ECU (brake actuator assembly) is connected to the combination meter assembly via CAN communication.

If any of the following is detected, the ABS warning light remains on:

·

The skid control ECU (brake actuator assembly) connectors are disconnected from the skid control ECU (brake actuator assembly).

·

There is a malfunction in the skid control ECU (brake actuator assembly) internal circuit.

·

There is an open in the harness between the combination meter assembly and the skid control ECU (brake actuator assembly).

·

The ABS control system is defective.

·

w/ Rear Differential Lock:

When the rear differential lock is in the lock state, the ABS is also prohibited and the ABS warning light turns on during operation.


HINT:

In some cases, the GTS cannot be used when the skid control ECU (brake actuator assembly) is abnormal.
