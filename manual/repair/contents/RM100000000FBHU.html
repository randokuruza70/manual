PROCEDURE

11165 1.SELECT COMBUSTION CHAMBER SHIM

a.

If using new combustion chamber sub-assembly:

i.

Using a micrometer, measure the thickness of each used combustion chamber sub-assembly at the position shown in the illustration, then select the number of combustion chamber shims to be used.

Select the Number of Combustion Chamber Shim:
Combustion chamber thickness No. of shims to be used
6.02 to 6.05 mm (0.2370 to 0.2382 in.) 0
6.06 to 6.08 mm (0.2386 to 0.2394 in.) 1
6.09 to 6.11 mm (0.2398 to 0.2406 in.) 2


Combustion Chamber Shim thickness:

0.03 mm (0.00118 in.)


NOTICE:

If combustion chamber shims were already being used, do not perform the above step, use the same number of combustion chamber shims as were used before.




b.

If reusing combustion chamber sub-assembly:

i.

Install the combustion chamber sub-assembly back in the same position it was originally at.



11106 2.INSTALL COMBUSTION CHAMBER SUB-ASSEMBLY

a.

Align the knock pin of the combustion chamber sub-assembly with the notch of the cylinder head sub-assembly.


b.

Using a plastic-faced hammer, tap in the combustion chamber sub-assembly.


c.

Using a dial indicator, measure the protrusion of the combustion chamber sub-assembly from the cylinder head sub-assembly.

Combustion chamber protrusion:

Minus 0.04 to Plus 0.04 mm (Minus 0.00157 to Plus 0.00157 in.)


If the protrusion is less than the specified, adjust with combustion chamber shims.

Combustion Chamber Shim thickness:

0.03 mm (0.00118 in.)


If the protrusion is more than the specified, replace the combustion chamber sub-assembly and recheck the protrusion.
