1GR-FE ENGINE CONTROL  SFI SYSTEM  P0504  Brake Switch "A" / "B" Correlation  


DESCRIPTION

The stop light switch assembly is a duplex system that transmits two signals: STP and ST1-. These two signals are used by the ECM to monitor whether or not the brake system is working properly. If the signals, which indicate the brake pedal is being depressed and released, are detected simultaneously, the ECM interprets this as a malfunction in the stop light switch assembly and stores the DTC.

HINT:

The normal conditions are as shown in the table below. The signals can be read using the GTS.

Signal (ECM Terminal) Brake Pedal Released In Transition Brake Pedal Depressed
STP OFF ON ON
ST1- ON ON OFF

·

[OFF] denotes ground potential.

·

[ON] denotes battery potential (+B).

·

On the Techstream, both the Data List items Stop Light Switch and ST1 are ON when the brake pedalis depressed because the ST1 indication characteristic is opposite to the Stop Light Switch indication.

CAUTION / NOTICE / HINT

NOTICE:

Inspect the fuses for circuits related to this system before performing the following inspection procedure.


HINT:
·

Read freeze frame data using the GTS. Freeze frame data records the engine condition when malfunctions are detected. When troubleshooting, freeze frame data can help determine if the vehicle was moving or stationary, if the engine was warmed up or not, if the air fuel ratio was lean or rich, and other data from the time the malfunction occurred.

·
