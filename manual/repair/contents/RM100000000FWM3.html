WIRELESS DOOR LOCK CONTROL SYSTEM DESCRIPTION


a.

The wireless door lock control system locks/unlocks the doors remotely. This system has the following features:

·

The door control with receiver ECU assembly performs the code identification process and operates the door lock control.

·

A transmitter is used and contains the following 2 switches: the door lock switch and door unlock switch.




FUNCTION OF MAIN COMPONENTS


Components Functions
Door control with receiver ECU assembly Receives weak electric waves (recognition codes and function codes) from the door control transmitter and changes the waves to code data.
Door control transmitter
·

Contains the lock and unlock switches.

·

Transmits the weak electric waves (recognition codes and function codes) to the door control with receiver ECU assembly.


Front door courtesy light switch assembly LH, RH
Rear door courtesy light switch assembly LH, RH*1
Back door courtesy light switch assembly*2
·

Located on each door.

·

Detects the door status (open or closed) and outputs data to the door control with receiver ECU assembly via the combination meter assembly.

·

Turns on when the door is opened and turns off when the door is closed.


Front door lock assembly LH*3, RH*4
·

Detects the door lock status (lock or unlock) and outputs data to the door control with receiver ECU assembly.

·

Operates the motor in response to signals from the door control with receiver ECU assembly and locks and unlocks the doors.


Front door lock assembly RH*3, LH*4
Rear door lock assembly LH, RH*1
Back door lock assembly*2 		Operates the motor in response to signals from the door control with receiver ECU assembly and locks and unlocks the doors.

*1: w/ Rear Door

*2: except Pickup

*3: for LHD

*4: for RHD



SYSTEM FUNCTION


a.

The wireless door lock control system has the following functions:

Function Outline
All door lock function Pressing the lock switch locks all the doors.
All door unlock function Pressing the unlock switch unlocks all the doors.
Answer-back function
·

When the doors are locked by a wireless operation, the hazard warning lights flash once.

·

When the doors are unlocked by a wireless operation, the hazard warning lights flash twice.


Automatic lock function If no doors are opened within 30 seconds after they are unlocked by the door control transmitter, all the doors (including the back door) lock again automatically.
