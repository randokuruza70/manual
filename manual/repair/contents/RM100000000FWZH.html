BRAKE (FRONT)  FRONT BRAKE  REASSEMBLY  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

47713 1.TEMPORARILY INSTALL FRONT DISC BRAKE BLEEDER PLUG

a.

Temporarily install the bleeder plug to the disc brake cylinder assembly LH.

HINT:

The bleeder plug will be tightened to a torque specification in the Bleed Brake Line procedures.



b.

Install the bleeder plug cap to the bleeder plug.


2.INSTALL PISTON SEAL

a.

Apply a light coat of lithium soap base glycol grease to 4 new piston seals.


b.

Install the 4 piston seals to the disc brake cylinder assembly LH.


47731 3.INSTALL FRONT DISC BRAKE PISTON

a.
0,0 0,0 false

Lithium soap base glycol grease


Apply lithium soap base glycol grease to the 4 front disc brake pistons and 4 new cylinder boots.


b.

Install the 4 cylinder boots to the front disc brake pistons.


c.

Install the 4 front disc brake pistons (with the boots) to the disc brake cylinder assembly LH.

NOTICE:

Do not forcibly install the front disc brake pistons to the disc brake cylinder assembly LH.



4.INSTALL CYLINDER BOOT


a.

Install the 4 cylinder boots to the disc brake cylinder assembly LH.


b.

Using a screwdriver, install the 4 set rings.

NOTICE:

Be careful not to damage the cylinder boot.
