DESCRIPTION

DTC No. Detection Item DTC Detection Condition Trouble Area MIL Memory
P0299 Turbocharger / Supercharger Underboost The pressure is 55 kPa below the Target Booster Pressure for 40 seconds at an engine speed of 2800 to 4400 rpm
(2 trip detection logic)
·

Turbocharger sub-assembly

·

Extremely low battery voltage (P0560 output simultaneously)

·

Disconnected pipe between turbocharger and intake manifold

·

Disconnected manifold absolute pressure sensor hose

·

Throttle valve stuck closed

·

Electric EGR control valve assembly does not close

·

Exhaust system or intake system modified (leading to increased pressure loss, etc.)

·

Manifold absolute pressure sensor

·

Injection system (reduction in fuel injection volume)

·

Fuel filter element sub-assembly clog

·

ECM


Comes on DTC stored

HINT:

If DTC P0299 is stored, the following symptoms may appear.

When the nozzle vane of the turbocharger is stuck open:

·

Lack of power

·

Vehicle surge or hesitation under light or medium load

CAUTION / NOTICE / HINT

HINT:

Read freeze frame data using the GTS. Freeze frame data records the engine condition when malfunctions are detected. When troubleshooting, freeze frame data can help determine if the vehicle was moving or stationary, if the engine was warmed up or not, and other data from the time the malfunction occurred.
