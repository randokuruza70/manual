PROCEDURE

1.INSTALL SPIDER BEARING

HINT:

Use the same procedure for both spiders.


a.

Apply MP grease to a new spider and new spider bearing.

NOTICE:

Do not apply too much grease.



b.

Install the grease fitting to the spider.

Torque:
6.3 N*m (64 kgf*cm, 56 in.*lbf)


c.

Install the spider to the propeller shaft.
