POWER DOOR LOCK CONTROL SYSTEM DESCRIPTION


a.

The power door lock system locks/unlocks all the doors. The power window regulator master switch assembly*1 or the door control switch*2 sends "lock/unlock" request signals to the door control relay assembly*3 or door control with receiver ECU assembly*4. Then, the door control relay assembly*3 or door control with receiver ECU assembly*4 sends these request signals to the lock motors in each door to lock/unlock all doors at once in response to the inputs.

Operating the driver's door lock using a mechanical key sends request signals to lock/unlock the doors to the door control relay assembly*3 or door control with receiver ECU assembly*4.

*1: w/ Power Window Control System

*2: w/o Power Window Control System

*3: w/o Wireless Door Lock Control System

*4: w/ Wireless Door Lock Control System
