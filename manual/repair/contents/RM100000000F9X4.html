27020 1.INSTALL GENERATOR ASSEMBLY

a.

Install the generator assembly with the 2 bolts.

Torque:
43 N*m (438 kgf*cm, 32 ft.*lbf)


b.

Connect the wire harness.

i.

Install the 2 wire harness clamp brackets with the 2 bolts.

Torque:
8.0 N*m (82 kgf*cm, 71 in.*lbf)


ii.

Attach the clamp.


iii.

Connect the wire to terminal B and install the nut.

Torque:
9.8 N*m (100 kgf*cm, 87 in.*lbf)


iv.

Install the terminal cap.


v.

Connect the connector to the generator assembly.
