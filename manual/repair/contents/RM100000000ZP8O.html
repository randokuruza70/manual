DESCRIPTION

This DTC indicates an abnormality in the ECM.

DTC No. Detection Item DTC Detection Condition Trouble Area
P0575 Cruise Control Input Circuit When either of the following conditions is met:
·

Cruise control input signal abnormal

·

ECM internal cancel circuit malfunction


ECM

HINT:

The ECM receives signals from each sensor to control all functions of the cruise control system. When a trouble code is detected, the fail-safe function remains on until the ignition switch is turned off.
