PROCEDURE

52134G 1.INSTALL FRONT BUMPER BRACKET LH

a.

Install the front bumper bracket LH with the 2 bolts.


52133F 2.INSTALL FRONT BUMPER BRACKET RH

HINT:

Use the same procedures described for the LH side.


52101B 3.INSTALL FRONT BUMPER REINFORCEMENT PLATE SUB-ASSEMBLY

a.

Install the front bumper reinforcement plate sub-assembly with the 2 bolts.


81482C 4.INSTALL FOG LIGHT COVER LH

a.

w/ Fog Light:

Install the fog light cover LH with the 2 bolts.


b.

w/o Fog Light:

Install the fog light cover LH with the 2 bolts.


81481E 5.INSTALL FOG LIGHT COVER RH

HINT:

Use the same procedures described for the LH side.


52016B 6.INSTALL FRONT BUMPER BRACKET SUB-ASSEMBLY LH

a.

Install the front bumper bracket sub-assembly LH with the 2 bolts and nut.


52015B 7.INSTALL FRONT BUMPER BRACKET SUB-ASSEMBLY RH

a.

Install the front bumper bracket sub-assembly RH with the 3 bolts and nut.


b.

Install the wire harness clamp bracket with the bolt.


52119A 8.INSTALL FRONT BUMPER COVER

a.

Install the front bumper cover.


b.

Connect the connectors and attach the clamps.


c.

Install the 6 bolts, 2 nuts and 5 clips.


81484A 9.INSTALL FOG LIGHT PROTECTOR LH (w/ Fog Light)

a.

Attach the 3 claws to install the fog light protector LH.


b.

Install the clip.


81483B 10.INSTALL FOG LIGHT PROTECTOR RH (w/ Fog Light)

HINT:

Use the same procedures described for the LH side.


53911 11.INSTALL FRONT VALANCE PANEL

a.

Attach the 4 claws of the 2 radiator grille clips to install the front valance panel.


b.

Install the 3 screws.
