CAUTION / NOTICE / HINT

HINT:
·

Use the same procedures for the RH and LH sides.

·

The procedures listed below are for the LH side.



PROCEDURE

53865 1.INSTALL FRONT WHEEL NO. 3 OPENING BRACKET

a.

Install the front wheel No. 3 opening bracket with the 2 screws.


53862 2.INSTALL FRONT WHEEL NO. 2 OPENING BRACKET

a.

Install the front wheel No. 2 opening bracket with the 2 screws.


53861 3.INSTALL FRONT WHEEL NO. 1 OPENING BRACKET

a.

Install the front wheel opening bracket with the 2 screws.


53851C 4.INSTALL FRONT WHEEL NO. 1 OPENING MOULDING PAD

HINT:

When installing the front wheel opening extension sub-assembly LH, heat the front wheel opening extension sub-assembly LH using a heat light.


Standard:
Item Temperature
Front Wheel Opening Extension Sub-assembly LH 20 to 30°C (68 to 86°F)


NOTICE:

Do not heat the front wheel opening extension sub-assembly LH excessively.


a.

If reusing the front wheel opening moulding sub-assembly LH:

i.

Using a heat light, front wheel opening extension sub-assembly LH.


ii.

Remove the double-sided tape from the front wheel opening extension sub-assembly LH.


iii.

Wipe off any tape adhesive residue with cleaner.

Install a new front wheel No. 1 opening moulding pad.

i.

Remove the peeling paper from the face of a new front wheel No. 1 opening moulding pad.


ii.

Attach the moulding pad to the moulding, as shown in the illustration.


iii.

Align the front wheel No. 1 opening moulding pad with the mark-off line of the front wheel opening extension sub-assembly LH, and then install it.

HINT:

Press the No. 1 side mudguard protector firmly to install it.



iv.

After installing the front wheel No. 1 opening moulding pad, remove the unnecessary part above the perforation.



53088A 5.INSTALL FRONT WHEEL OPENING EXTENSION SUB-ASSEMBLY LH

a.

Install 8 new clips to the front wheel opening extension sub-assembly LH.


b.

Attach the 8 clips to install the front wheel opening extension sub-assembly LH.


c.

Install the 3 screws.


d.

Install a new grommet.


e.

Install 2 new rivets.

i.

Using an air riveter or hand riveter with a nose piece, install the front wheel opening extension sub-assembly LH with 2 new rivets.

HINT:

If the rivet cannot be cut, pull it once and cut it.
