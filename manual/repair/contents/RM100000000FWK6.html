BRAKE SYSTEM (OTHER)  VACUUM PUMP(for 1HZ)  REASSEMBLY  


PROCEDURE

1.INSTALL BEARING

a.

Using a press, press in a new bearing to the shaft.


2.INSTALL VACUUM PUMP SHAFT

a.
0.667,0.25 0.885,0.25 0.885,0.25 1.625,0.656 true 0.375,0.167 0.833,0.365 0.458,0.198 10 false SST

Using SST and a press, press in the shaft to the housing.

SST
09608-04031  


3.INSTALL HOLE SNAP RING

a.

Using snap ring pliers, install a new snap ring.

NOTICE:

Make sure that the snap ring is firmly installed to the groove.



4.INSTALL VACUUM PUMP GEAR

a.

Install the woodruff key and gear to the pump with the lock nut.

Torque:
110.3 N*m (1125 kgf*cm, 81 ft.*lbf)


5.INSTALL VACUUM PUMP ROTOR SUB-ASSEMBLY

a.

Apply engine oil to the rotor and install it to the shaft.


6.INSTALL VACUUM PUMP BLADE

a.

Install 5 new blades with the round end facing outward.

HINT:
·

Apply engine oil to the blade.

·

Make sure that the blades are flush with the rotor surface.




7.INSTALL VACUUM PUMP END COVER

a.

Install a new O-ring.


b.

Install the end cover and temporarily install the 3 bolts.


c.

Using a 5 mm pin punch and hammer, tap in the 2 straight pins.


d.

Tighten the 3 bolts.

Torque:
7.9 N*m (81 kgf*cm, 70 in.*lbf)


8.INSTALL CHECK VALVE SUB-ASSEMBLY

a.

Install a new gasket and the check valve sub-assembly.

Torque:
73.5 N*m (749 kgf*cm, 54 ft.*lbf)


9.INSTALL VACUUM HOSE UNION

a.

Install 2 new gaskets and the union with the union bolt.

Torque:
13.7 N*m (140 kgf*cm, 10 ft.*lbf)

HINT:

Align and insert the union pin into the matching hole of the housing.
