DOOR LOCK  FRONT DOOR LOCK  INSTALLATION  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for RHD and LHD vehicles.

·

The procedure listed below is for LHD vehicles.

·

Use the same procedure for the RH and LH side.

·

The procedure listed below is for the LH side.



PROCEDURE

69120A 1.INSTALL FRONT DOOR LOCK ACTUATOR ASSEMBLY LH (w/ Power Door Lock, Type B)

a.

Attach the guide in the direction indicated by the arrow in the illustration.


b.

Install the front door lock actuator assembly LH to the front door lock assembly LH with the 2 screws.


69320 2.INSTALL FRONT DOOR LOCK ASSEMBLY LH

a.

Apply MP grease to the sliding parts of the front door lock assembly LH.


b.

Make sure that the front door lock open rod is securely connected to the front door lock assembly LH.


c.

Using a T30 "TORX" socket wrench, install the front door lock assembly LH with the 3 screws.

Torque:
5.0 N*m (51 kgf*cm, 44 in.*lbf)


d.

Attach the 2 snaps and connect the 2 control links as shown in the illustration.


e.

w/ Power Door Lock Type A:

i.

Connect the connector and attach the clamp.



f.

w/ Power Door Lock Type B:

i.

Connect the connector and attach the clamp.


ii.

Install the Bolt.

Torque:
5.0 N*m (51 kgf*cm, 44 in.*lbf)



69310 3.INSTALL FRONT DOOR LOCK ASSEMBLY RH (w/ Double Inside Handles)

a.

Apply MP grease to the sliding parts of the front door lock assembly RH.


b.

Using a T30 "TORX" socket wrench, install the front door lock assembly RH with the 3 screws.

Torque:
5.0 N*m (51 kgf*cm, 44 in.*lbf)


c.

Attach the 2 snaps and connect the 2 control links as shown in the illustration.


d.

w/ Power Door Lock:

i.

Connect the connector.


ii.

Attach the clamp.


iii.

Install the Bolt.

Torque:
5.0 N*m (51 kgf*cm, 44 in.*lbf)
