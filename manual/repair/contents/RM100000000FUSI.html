RM25T0E _53 Suspension _029910 REAR SUSPENSION _0138030 REAR LEAF SPRING A

REAR SUSPENSION  REAR LEAF SPRING  REASSEMBLY  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

48220 1.REASSEMBLY REAR SPRING ASSEMBLY LH

a.

Install the rear spring silencers to the rear leaf spring.


b.

Install the center bolt and nut.

Torque:
49 N*m (500 kgf*cm, 36 ft.*lbf)


c.

Install the hexagon bolt, washer and rear spring clip bolt case to the rear leaf spring.

Torque:
19 N*m (194 kgf*cm, 14 ft.*lbf)


d.

Using a vise, caulk the clip.


e.

Check that there is no gap between the spring clip and spring.


48210B 2.INSTALL REAR SPRING PIN BUSH (for 1VD-FTV 76, 78 Series)

a.
2.333,0.813 2.844,0.813 2.844,0.813 2.323,1.51 true 2.917,0.74 3.302,0.948 0.385,0.208 10 false SST

Using SST and a press, press in a new bush (outer side).

b.

Using SST and a press, press in a new bush (inner side).

48210Q 3.INSTALL REAR SPRING SHACKLE BUSH (except 1VD-FTV 76, 78 Series)

a.

Install the 2 bushes to the rear spring.


48375 4.INSTALL LEAF SPRING DYNAMIC DAMPER (w/ Dynamic Damper)

a.

Install the dynamic damper with the bolt and nut.

Torque:
19 N*m (194 kgf*cm, 14 ft.*lbf)
