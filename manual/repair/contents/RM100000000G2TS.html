LIGHTING (EXT)  LICENSE PLATE LIGHT ASSEMBLY(for Van)  INSTALLATION  


PROCEDURE

81270 1.INSTALL LICENSE PLATE LIGHT ASSEMBLY (for Back Door)

a.

Pass the wire harness through the back door and connect the grommet.


b.

Install the license plate light assembly with the 3 nuts.

Torque:
3.2 N*m (33 kgf*cm, 28 in.*lbf)


c.

Connect the connector.


81270 2.INSTALL LICENSE PLATE LIGHT ASSEMBLY (w/ Auxiliary Fuel Tank)

a.

Install the bulb socket of the license plate light assembly with the 2 nuts.

HINT:

Use the same procedure to install the bulb socket of the license plate light assembly on the other side.



b.

Attach the 2 wire harness clamps.


c.

Pass the wire harness through the back door and connect the grommet.


d.

Connect the connector.


81690 3.INSTALL LICENSE PLATE LIGHT COVER SUB-ASSEMBLY (w/ Auxiliary Fuel Tank)

a.

Install the license plate light cover sub-assembly with the 2 screws.

HINT:

Use the same procedure to install the license plate light cover sub-assembly on the other side.
