CRUISE CONTROL  CRUISE CONTROL SYSTEM  P0503  Vehicle Speed Sensor "A" Intermittent / Erratic / High  


DESCRIPTION

The skid control ECU (brake actuator assembly) outputs the vehicle speed signal to the ECM through the combination meter assembly.

DTC No. Detection Item DTC Detection Condition Trouble Area
P0503 Vehicle Speed Sensor "A" Intermittent / Erratic / High Cruise control in operation: When the ECM receives sudden changes in the vehicle speed signal
·

Vehicle stability control system

·

Meter / gauge system

·

ECM



CAUTION / NOTICE / HINT

·

This DTC may be output by the vehicle stability control system at the same time. In that case, perform troubleshooting for the vehicle stability control system first.

·

When replacing the combination meter assembly, make sure to replace it with a new one.
