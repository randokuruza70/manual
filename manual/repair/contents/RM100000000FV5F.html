CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

47629 1.INSTALL PARKING BRAKE BELLCRANK BRACKET

a.

Install the parking brake bellcrank bracket and pin to the parking brake bell crank LH with a new C-washer as shown in the illustration.


47634 2.INSTALL PARKING BRAKE BELLCRANK BOOT

a.
0,0 0,0 false

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to the parking brake bellcrank boot.

HINT:

Apply at least 0.2 to 0.25 g (0.007 to 0.008 oz.) of lithium soap base glycol grease to the parking brake bellcrank boot.



b.

Install the parking brake bellcrank boot to the parking brake bellcrank bracket.


47639A 3.INSTALL PARKING BRAKE BELLCRANK STOPPER BOLT

a.

Install the parking brake bellcrank stopper bolt and nut to the parking brake bell crank LH.

HINT:

The nut is tightened in a later step when adjusting the parking brake lever travel.



47625B 4.INSTALL PARKING BRAKE BELL CRANK LH

a.

Install the parking brake bell crank LH with the 2 bolts.

Torque:
12.5 N*m (127 kgf*cm, 9 ft.*lbf)


47623B 5.INSTALL PARKING BRAKE BELL CRANK RETURN TENSION SPRING

a.

Install the 2 parking brake bell crank return tension springs.


46410 6.CONNECT NO. 1 PARKING BRAKE CABLE ASSEMBLY

a.

Connect the No. 1 parking brake cable assembly with the pin and a new clip to the parking brake bell crank LH.


47616A 7.INSTALL PARKING BRAKE WIRE

a.

Install the parking brake wire with the pin and a new C-washer to the parking brake bell crank LH.


47447E 8.INSTALL PARKING BRAKE SHOE HOLD DOWN SPRING PIN (for Front Side)

a.

Install the parking brake shoe hold down spring pin.


47447F 9.INSTALL PARKING BRAKE SHOE HOLD DOWN SPRING PIN (for Rear Side)

a.

Install the parking brake shoe hold down spring pin.


47459B 10.INSTALL PARKING BRAKE SHOE GUIDE PLATE SET BOLT

a.

Install the parking brake shoe guide plate set bolt and parking brake shoe guide plate.

Torque:
18 N*m (184 kgf*cm, 13 ft.*lbf)


11.APPLY HIGH-TEMPERATURE GREASE

a.
0,0 0,0 false

High-temperature grease


Apply a light coat of high-temperature grease to the areas of the backing plate which make contact with the shoe as shown in the illustration.


47612 12.INSTALL PARKING BRAKE SHOE LEVER LH

a.

Apply a light coat of high-temperature grease to the areas of the parking brake shoe lever LH which make contact with the No. 1 parking brake shoe assembly LH.


b.

Using a feeler gauge, measure the clearance between the No. 1 parking brake shoe assembly LH and parking brake shoe lever LH.

Standard clearance:

Less than 0.25 mm (0.00984 in.)


If the clearance is not within the specification, replace the shim with one of a different thickness so that the clearance is within the specification.

Standard Shim:
Shim Thickness Shim Thickness
0.3 mm (0.0118 in.) 0.6 mm (0.0236 in.)
0.4 mm (0.0157 in.) 0.9 mm (0.0354 in.)
0.5 mm (0.0197 in.) -



c.

Install the parking brake shoe lever and shim to the No. 1 parking brake shoe assembly LH with a new C-washer as shown in the illustration.


47616A 13.CONNECT PARKING BRAKE WIRE

a.

Using needle-nose pliers, connect the parking brake wire to the parking brake shoe lever LH as shown in the illustration.

NOTICE:

Be careful not to damage the parking brake wire.



46550A 14.INSTALL NO. 3 PARKING BRAKE SHOE RETURN TENSION SPRING

a.

Install the No. 3 parking brake shoe return tension spring to the No. 1 parking brake shoe assembly LH.


b.

Install the No. 2 parking brake shoe assembly LH to the No. 3 parking brake shoe return tension spring.


47406A 15.INSTALL PARKING BRAKE SHOE ADJUSTING SCREW SET

a.
1.469,2.323 1.625,2.281 false

High-temperature grease


Apply a light coat of high-temperature grease to the areas of the shoe adjusting screw set shown in the illustration.


b.

Install the parking brake shoe adjusting screw set.


46580 16.INSTALL NO. 1 PARKING BRAKE SHOE ASSEMBLY LH

a.

Pull the No. 1 parking brake shoe assembly LH and No. 2 parking brake shoe assembly LH open by hand, and then align them with the backing plate.


b.

Install the No. 1 parking brake shoe assembly LH to the backing plate with the 2 parking brake shoe hold down spring cups and parking brake shoe hold down spring.


47614 17.INSTALL PARKING BRAKE SHOE STRUT LH

a.

Install the parking brake shoe strut and parking brake shoe strut compression spring.


b.

Install the No. 2 parking brake shoe assembly LH in the correct position.


46590 18.INSTALL NO. 2 PARKING BRAKE SHOE ASSEMBLY LH

a.

Install the No. 2 parking brake shoe assembly LH to the backing plate with the 2 parking brake shoe hold down spring cups and parking brake shoe hold down spring.


46550E 19.INSTALL NO. 2 PARKING BRAKE SHOE RETURN TENSION SPRING

a.

Install the 2 No. 2 parking brake shoe return tension springs.


46550G 20.INSTALL NO. 1 PARKING BRAKE SHOE RETURN TENSION SPRING

a.

Install the No. 1 parking brake shoe return tension spring.
