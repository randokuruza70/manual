1VD-FTV ENGINE CONTROL  CAMSHAFT POSITION SENSOR  INSTALLATION  


PROCEDURE

11301M 1.INSTALL CAMSHAFT POSITION SENSOR


a.

Apply a light coat of engine oil to the O-ring of the camshaft position sensor.

HINT:
·

When reusing the camshaft position sensor, inspect the O-ring.

·

If the O-ring has scratches or cuts, replace the camshaft position sensor.




b.

Install the camshaft position sensor with the bolt to the timing chain cover sub-assembly.

Torque:
5.0 N*m (51 kgf*cm, 44 in.*lbf)


c.

Connect the connector to the camshaft position sensor.
