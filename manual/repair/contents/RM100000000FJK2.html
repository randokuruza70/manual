HINT:
·

The cylinder, which has the compensation value for Injection Feedback Val #1, #2, #3, #4, #5, #6, #7 and #8 that varies from other cylinders, may be malfunctioning.

·

If DTC P0093 (fuel leaks from high-pressure area) is set simultaneously, a leakage malfunction of the cylinder has probably occurred.

·

If DTC P062D (open or short in the No. 1 injector driver (EDU) or injector circuit) is set simultaneously, there is probably a faulty circuit located between the ECM and the injector including the No. 1 EDU.

·

If DTC P062E (open or short in the No. 2 injector driver (EDU) or injector circuit) is set simultaneously, there is probably a faulty circuit located between the ECM and the injector including the No. 2 EDU.

·

If only DTC P1238 (injector malfunction) is present, an injector mechanical malfunction or insufficient compression (valve clearance, etc.) has probably occurred. An open or short malfunction of the ECM, EDU or injector circuit is unlikely.

MONITOR DESCRIPTION

P1238 (Injection malfunction, except open or short in injector circuit):

This DTC will be set if the engine speed fluctuation between each cylinder is large. The ECM monitors changes in the crankshaft rotation speed using the crankshaft position sensor in order to detect improper combustion. The camshaft position sensor also plays a role in determining which cylinder has the problem. The fluctuation counter increases when irregular crankshaft rotation speed variation exceeds the threshold with the engine idling. Therefore, if any one of the cylinders is operating poorly (rough idle), the ECM sets this DTC.
