PROCEDURE

31477 1.INSTALL RELEASE CYLINDER BLEEDER PLUG

a.

Install the release cylinder bleeder plug to the clutch release cylinder body.

Torque:
10.8 N*m (110 kgf*cm, 8 ft.*lbf)


b.

Install the bleeder plug cap.


04313 2.INSTALL CLUTCH RELEASE CYLINDER KIT

a.

Install 2 new cylinder cups onto a new piston.


b.

Install a new spring to the piston.


c.
0,0 0,0 false

Lithium soap base glycol grease


Apply lithium soap base glycol grease to the portions indicated by the arrows in the illustration.


d.

Install the piston with the spring to the clutch release cylinder body.

NOTICE:

Do not damage the inside of the cylinder body.



e.

Install the push rod to a new boot.


f.

Install the boot with the push rod to the clutch release cylinder body.
