CRUISE CONTROL  CRUISE CONTROL SYSTEM  P0500  Vehicle Speed Sensor Malfunction  


DESCRIPTION

The speed sensor installed on the wheels outputs wheel speed pulse signals which are then converted to vehicle speed pulse signals by the skid control ECU (brake actuator assembly). These pulse signals are then output to the combination meter assembly as vehicle speed signals and then input to the ECU.

DTC No. Detection Item DTC Detection Condition Trouble Area
P0500 Vehicle Speed Sensor Malfunction The vehicle speed signal from the vehicle speed sensor is cut for 0.14 seconds or more while the cruise control is in operation ECD system

CAUTION / NOTICE / HINT

This DTC may be output by the ECD system at the same time. In that case, perform troubleshooting for the ECD system first.
