BF1B TRANSFER / 4WD / AWD  TRANSFER SHIFT LEVER ASSEMBLY  INSTALLATION  


PROCEDURE

36321 1.INSTALL FRONT TRANSFER DRIVE SHIFT LEVER (for Direct Shift Type)

a.

Install a new bush to the front transfer drive shift lever.


b.

Install the front transfer drive shift lever and 2 washers and then install a new clip.


c.

Install the transfer shift lever shaft and wave washer.


d.

Install the 2 washers and nut.

Torque:
44.1 N*m (450 kgf*cm, 33 ft.*lbf)


36323 2.INSTALL TRANSFER SHIFT LEVER GUIDE (for Direct Shift Type)

a.

Install the front drive shift link lever bush.


b.

Install the transfer shift lever guide with the 2 bolts.

Torque:
39.2 N*m (400 kgf*cm, 29 ft.*lbf)


c.

Install the nut, washer and wave washer.

Torque:
24.5 N*m (250 kgf*cm, 18 ft.*lbf)


d.

Install the manual transmission assembly.

36322 3.INSTALL TRANSFER HIGH AND LOW SHIFT LEVER (for Vacuum Shift Type)

a.

Install the 2 transfer shift lever bush, high and low shift rod grommet and high and low shift rod spacer to the transfer high and low shift lever.


b.

Install the high and low shift rod pin and transfer high and low shift lever and then Install a new clip.


c.

Install the nut, wave washer and washer.

Torque:
24.5 N*m (250 kgf*cm, 18 ft.*lbf)
