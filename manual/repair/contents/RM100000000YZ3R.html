1VD-FTV ENGINE CONTROL  ECD SYSTEM(w/ DPF)  P1605  Rough Idling  


DESCRIPTION

This DTC is stored when the speed of a specific cylinder becomes low and the idle speed becomes unstable. This DTC is also stored if the vehicle runs out of fuel. Be sure to check whether there is enough fuel in the fuel tank before inspection.

DTC No. Detection Item DTC Detection Condition Trouble Area MIL Memory
P1605 Rough Idling When the accelerator pedal is fully closed and the engine is idling, the speed of a specific cylinder is low for 2 to 10 seconds (1 trip detection logic).

CAUTION / NOTICE / HINT

In order for idling to be stable, glow system operation, fuel system operation and compression sufficient to maintain the target idling speed are necessary.

HINT:
·

In contrast to normal malfunction diagnosis for components, circuits and systems, DTCs P1603 and P1605 are used to determine the malfunctioning area from the problem symptoms and freeze frame data when the user mentions problems such as engine stall.

As these DTCs can be stored as a result of certain user actions, even if these DTCs are output, if the customer makes no mention of problems, clear these DTCs without performing any troubleshooting and return the vehicle to the customer.

·

If any other DTCs are output, perform troubleshooting for those DTCs first.

·

Read freeze frame data using the GTS. Freeze frame data records the engine condition when malfunctions are detected. When troubleshooting, freeze frame data can help determine if the vehicle was moving or stationary, if the engine was warmed up or not, and other data from the time the malfunction occurred.

·

When confirming the freeze frame data, be sure to check all 5 sets of freeze frame data.

3.DETERMINE CAUSE OF PROBLEM (CHECK FREEZE FRAME DATA)
a.

Determine the cause of the problem based on the freeze frame data recorded in Procedure 1.


Rough Idle #1 to #8:
Judgment of Data List Values Problem Cause Diagnosis Note
Any of Rough Idle #1 to #8 in the freeze frame data is ON Fuel injector If the cylinder that displays Rough Idle ON also has a value of more than 3.0 mm3/st for Injection Feedback Val, there may be improper combustion in that cylinder.
HINT:

When there are temporary changes in the injection characteristics, the malfunction may not be indicated by the value for Injection Feedback Val in the freeze frame data.
