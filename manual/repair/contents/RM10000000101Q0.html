RM25T0E _58 Vehicle Interior _029927 METER / GAUGE / DISPLAY _0137559 METER / GAUGE SYSTEM J

METER / GAUGE / DISPLAY  METER / GAUGE SYSTEM  Tachometer Malfunction  


DESCRIPTION

w/ VSC


In this circuit, the combination meter assembly receives engine speed signals from the ECM via the CAN communication system. The combination meter assembly displays the engine speed calculated based on the data received from the ECM.


w/o VSC


The combination meter receives revolution signals from the ECM*1, emission control ECU*2 or pickup sensor*3. The combination meter displays engine speed data that is calculated based on the data received from the ECM*1, emission control ECU*2 or pickup sensor*3.

*1: for 1GR-FE, 1VD-FTV

*2: for 1HZ (w/ EGR System)

*3: for 1HZ (w/o EGR System)
