SUPPLEMENTAL RESTRAINT SYSTEMS  AIRBAG SYSTEM  B1625/22  Side Airbag Sensor LH Circuit Malfunction  


DESCRIPTION

The door side airbag sensor LH consists of a diagnostic circuit and sideal deceleration sensor, etc.

If the airbag sensor assembly receives signals from the sideal deceleration sensor, it determines whether the SRS should be activated.

DTC B1625/22 is stored when a malfunction is detected in the door side airbag sensor LH circuit.

DTC No. Detection Item DTC Detection Condition Trouble Area
B1625/22 Side Airbag Sensor LH Circuit Malfunction Either condition is met:
·

A door side airbag sensor LH malfunction.

·

An airbag sensor assembly malfunction.


·

Door side airbag sensor LH

·

Airbag sensor assembly
