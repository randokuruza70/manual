1GR-FE FUEL  FUEL TANK SELECT VALVE(for Fuel Sub Tank)  INSTALLATION  


PROCEDURE

77720E 1.INSTALL FUEL TANK SOLENOID RETURN VALVE ASSEMBLY

a.

Install the fuel tank solenoid return valve assembly with the bolt to the fuel tube protector sub-assembly.

Torque:
12.5 N*m (127 kgf*cm, 9 ft.*lbf)


b.

Attach the wire harness clamp.


77710B 2.INSTALL FUEL TANK SOLENOID MAIN VALVE ASSEMBLY

a.

Install the fuel tank solenoid main valve assembly with the bolt to the fuel tube protector sub-assembly.

Torque:
8.0 N*m (82 kgf*cm, 71 in.*lbf)


77406 3.INSTALL FUEL TANK SOLENOID VALVE WITH BRACKET

a.

Install the fuel tank solenoid valve with bracket with the 3 bolts.

Torque:
30 N*m (306 kgf*cm, 22 ft.*lbf)


b.

Connect the fuel tank solenoid return valve connector.


c.

Connect the 3 fuel tubes to the No. 3 fuel main tube and fuel tank solenoid main valve assembly.

d.

Connect the 3 fuel hoses to the fuel tank solenoid return valve assembly and slide the 3 clamps to secure the hose.


e.

f.

Attach the wire harness clamp.


77268A 4.INSTALL NO. 3 FUEL TUBE PROTECTOR

a.

Install the No. 3 fuel tube protector with the 2 bolts to the fuel tank solenoid valve with bracket.

Torque:
12.5 N*m (127 kgf*cm, 9 ft.*lbf)
