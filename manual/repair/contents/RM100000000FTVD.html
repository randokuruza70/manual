RM25T0E _58 Vehicle Interior _029469 HEATING / AIR CONDITIONING _0137770 CONDENSER(for 1GR-FE) A

HEATING / AIR CONDITIONING  CONDENSER(for 1GR-FE)  REASSEMBLY  


PROCEDURE

88467 1.INSTALL PIPING BRACKET LH

a.

Install the piping bracket LH with the 2 bolts.

Torque:
5.4 N*m (55 kgf*cm, 48 in.*lbf)


88467 2.INSTALL PIPING BRACKET RH

a.

Install the piping bracket RH with the 2 bolts.

Torque:
5.4 N*m (55 kgf*cm, 48 in.*lbf)


88467B 3.INSTALL LOWER COOLER BRACKET LH

a.

Install the lower cooler bracket LH with the bolt.

Torque:
5.4 N*m (55 kgf*cm, 48 in.*lbf)


88467B 4.INSTALL LOWER COOLER BRACKET RH

a.

Install the lower cooler bracket RH with the bolt.

Torque:
5.4 N*m (55 kgf*cm, 48 in.*lbf)


88467B 5.INSTALL UPPER COOLER BRACKET LH

a.

Install the upper cooler bracket LH with the bolt.

Torque:
4.4 N*m (45 kgf*cm, 39 in.*lbf)


88467B 6.INSTALL UPPER COOLER BRACKET RH

a.

Install the upper cooler bracket RH with the bolt.

Torque:
4.4 N*m (45 kgf*cm, 39 in.*lbf)


88716A 7.INSTALL COOLER REFRIGERANT LIQUID PIPE A


a.

Remove the attached vinyl tape from the tube and the connecting part of the cooler condenser assembly.


b.

Sufficiently apply compressor oil to a new O-ring and the fitting surface of the tube joint.

Compressor oil:

ND-OIL 8 or equivalent



c.

Install the O-ring on the cooler refrigerant liquid pipe A.


d.

Install the cooler refrigerant liquid pipe A on the cooler condenser with the bolt.

Torque:
5.4 N*m (55 kgf*cm, 48 in.*lbf)

NOTICE:
·

When tightening the bolt, do not allow any tools to contact the pipe.

·

When tightening the bolt, hold a part of the pipe near the connector.




88474 8.INSTALL COOLER DRYER

a.

Using pliers, install the cooler dryer.


b.

Apply a sufficient amount of compressor oil to the contact surfaces of a new O-ring and the cap.

Compressor oil:

ND-OIL 8 or equivalent



c.

Install the O-ring to the cap.


d.

Using a 14 mm socket hexagon wrench, install the cap to the modulator.

Torque:
2.9 N*m (30 kgf*cm, 26 in.*lbf)
