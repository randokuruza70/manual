WINDOW / GLASS  BACK WINDOW GLASS  REASSEMBLY  


PROCEDURE

56111 1.CLEAN WINDSHIELD GLASS

a.

Clean the outer circumference of the glass with non-residue solvent.

NOTICE:
·

Do not touch the glass surface after cleaning it.

·

Be careful not to damage the glass.

·

Even if using new glass, clean the glass with non-residue solvent.




64829 2.INSTALL BACK WINDOW FRAME FILM (for Slide Type)

a.

Install the 6 back window frame films to the back window frame.


64826 3.INSTALL LOWER BACK WINDOW CHANNEL RUN (for Slide Type)

a.

Install the lower back window channel run to the back window frame.


64825 4.INSTALL UPPER BACK WINDOW CHANNEL RUN (for Slide Type)

a.

Install the upper back window channel run to the back window frame.


64812C 5.INSTALL BACK WINDOW GLASS WEATHERSTRIP (for Slide Type)

a.

Install the back window glass weatherstrip.


64812C 6.INSTALL BACK WINDOW GLASS WEATHERSTRIP (for Slide Type)

a.

Align the grooves on the back window frame and back window glass weatherstrip to install.
