RM25T0E _58 Vehicle Interior _029929 SUPPLEMENTAL RESTRAINT SYSTEMS _0138307 AIRBAG SYSTEM D

SUPPLEMENTAL RESTRAINT SYSTEMS  AIRBAG SYSTEM  CHECK MODE PROCEDURE  


NOTICE:

Enter "Signal Check" from the "DTC Check" screen displayed on the GTS to clear the output DTCs (both present and past).


HINT:
·

DTCs can be stored more sensitively in check mode than in normal diagnosis mode.

·

Perform the check mode inspection when a malfunction in each squib circuit is suspected even after the normal system code is output through normal diagnosis mode inspection.



CHECK MODE PROCEDURE (Using the GTS)


a.

Turn the ignition switch off.


b.

Connect the GTS to the DLC3.


c.

Turn the ignition switch to ON.


d.

Turn the GTS on.


e.

Select "Signal Check" and proceed with checking using the GTS.
