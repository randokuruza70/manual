1HZ FUEL  FUEL TANK SELECT VALVE  INSTALLATION  


PROCEDURE

77720E 1.INSTALL FUEL TANK SOLENOID RETURN VALVE ASSEMBLY

a.

Install the fuel tank solenoid return valve assembly with the bolt to the fuel solenoid valve main bracket.

Torque:
12.5 N*m (127 kgf*cm, 9 ft.*lbf)


b.

Attach the wire harness clamp.


77710B 2.INSTALL FUEL TANK SOLENOID MAIN VALVE ASSEMBLY

a.

Install the fuel tank solenoid main valve assembly with the bolt to the fuel solenoid valve main bracket.

Torque:
12.5 N*m (127 kgf*cm, 9 ft.*lbf)


77406 3.INSTALL FUEL TANK SOLENOID VALVE WITH BRACKET

a.

Install the fuel tank solenoid valve with bracket with the 2 bolts.

Torque:
30 N*m (306 kgf*cm, 22 ft.*lbf)


b.

Connect the fuel tank solenoid return valve connector.


c.

Connect the 3 fuel hoses to the fuel tank solenoid main valve assembly and slide the 3 clamps to secure the hoses.


d.

Connect the 3 fuel hoses to the fuel tank solenoid return valve assembly and slide the 3 clamps to secure the hoses.


e.

Attach the wire harness clamp.
