1GR-FE ENGINE CONTROL  CAMSHAFT OIL CONTROL VALVE  ON-VEHICLE INSPECTION  


PROCEDURE

15330B 1.CHECK CAMSHAFT TIMING OIL CONTROL VALVE ASSEMBLY

a.

Connect the GTS to the DLC3.


b.

Start the engine and turn the GTS on.


c.

Warm up the engine.


d.

Enter the following menus: Powertrain / Engine and ECT / Active Test / Control the VVT System (Bank 1) or Control the VVT System (Bank 2).

Powertrain > Engine and ECT > Active Test
Tester Display
Control the VVT System (Bank1)
50004 61 112
Powertrain > Engine and ECT > Active Test
Tester Display
Control the VVT System (Bank2)
50004 61 113

e.

Check the engine speed while operating the oil control valve (OCV) using the GTS.

OK:
Tester Operation Specified Condition
VVT system is OFF (OCV is OFF) Normal engine speed
VVT system is ON (OCV is ON) Rough idle or engine stalled


If the result is not as specified, check the camshaft timing oil control valve assembly, wiring and camshaft timing gear assembly.
