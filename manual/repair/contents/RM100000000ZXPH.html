41331 2.INSPECT REAR DIFFERENTIAL SIDE GEAR

a.

Install the rear No. 2 differential side gear thrust washers, clutch plates and rear differential side gear thrust shim to the rear differential side gear.

HINT:

When installing the rear differential side gear thrust shim, make sure the surface with no oil groove faces the differential case side.



b.

Install the rear differential side gear to the differential case.


c.

Install the 4 rear differential pinions and 4 rear differential pinion thrust washers to the rear differential spider.


d.

Align the spring retainer holes with the straight pins and install the spring retainer.


e.

Install the spider assembly to the differential case.

HINT:

Install the spider assembly to the differential case tightly and do not move the spring retainer.



f.

Using the dial indicator, measure the side gear backlash while holding the rear differential side gear and rear differential spider.

Standard backlash:

0.02 to 0.15 mm (0.00079 to 0.00590 in.)


HINT:
·

Measure at all 4 locations.

·

Measure the backlash at the differential case LH and RH.

·

Apply hypoid gear oil LSD to each sliding surface and rotating part.



If the backlash is not within the specification, select the proper rear differential side gear thrust shim.


3.ASSEMBLE DIFFERENTIAL CASE

a.

Install the spider assembly to the differential case LH.

HINT:

Install the spider assembly to the differential case LH tightly and do not move the spring retainer.



b.

Install the compression spring and spring retainer RH.


c.

Install the side gear assembly RH.

NOTICE:
·

Be careful not to drop the side gear.

·

Check the pinion and side gear alignment.




d.

Align the matchmarks and assemble the differential case LH and RH.


e.

Install the 8 bolts and tighten them uniformly, a little at a time.

Torque:
47.1 N*m (480 kgf*cm, 35 ft.*lbf)

5.INSTALL DIFFERENTIAL RING GEAR

a.

Clean the threads of the bolts and differential case with non-residue solvent.


b.

Clean the contact surface of the differential case and differential ring gear.


c.

Heat the differential ring gear to about 100°C (212°F) in boiling water.


d.

Carefully take the differential ring gear out of the boiling water.


e.

After the moisture on the differential ring gear has completely evaporated, quickly install the differential ring gear to the differential case.


f.
1.24,1.313 2.167,1.208 2.167,1.208 1.219,1.458 true 2.208,1.125 2.969,1.292 0.76,0.167 10 false *a
*a Matchmark


Align the matchmarks on the differential ring gear and differential case.


g.

Apply thread lock adhesive to the threads of the 12 set bolts and temporarily install them.

Thread lock:

Toyota Genuine Adhesive 1360 K, Three Bond 1360 K or equivalent



h.

After the differential ring gear cools down sufficiently, tighten the set bolts.

Torque:
137 N*m (1397 kgf*cm, 101 ft.*lbf)


6.INSPECT DIFFERENTIAL RING GEAR RUNOUT

a.

Install the differential case and 2 rear differential case bearings (outer race) to the differential carrier, and install the 2 rear differential bearing adjusting nuts so that there is no play in the bearing.


b.

Install the 2 bearing caps with the 4 bolts.

Torque:
90 N*m (918 kgf*cm, 66 ft.*lbf)


c.

Using a dial indicator, check the ring runout.

Maximum runout:

0.05 mm (0.00197 in.)


If the runout is more than the maximum, replace the differential ring gear.


d.

Remove the 4 bolts, 2 bearing caps, 2 rear differential bearing adjusting nuts and rear differential case.
