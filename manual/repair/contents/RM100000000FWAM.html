THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  Security Indicator Light Does not Blink  


DESCRIPTION

The transponder key ECU assembly blinks the security indicator light combination meter assembly when the immobiliser is set (no key is in the ignition key cylinder).
