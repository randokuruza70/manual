POWER WINDOW CONTROL SYSTEM DESCRIPTION


a.

The power window control system controls the power window operation using the power window regulator motors. The main controls of this system are the power window regulator master switch assembly (mounted on the driver door), power window regulator switch assembly (mounted on the front passenger door) and power window regulator switch assemblies (mounted on the rear doors) (w/ Rear Power Window). Operating a window switch results in electrical power being transmitted to the corresponding power window regulator motor assembly.



FUNCTION OF MAIN COMPONENTS


The power window control system consists of the following components:

Component Outline
Power window regulator master switch assembly
·

Controls power window operation for all windows.

·

When the window lock switch is set to the lock position, window operation will be possible for all windows.


Power window regulator switch assembly (for Front Passenger Door)
·

Located on the front passenger door.

·

Controls front passenger window operation.


·

Power window regulator switch assembly (for Rear Door LH, w/ Rear Power Window)

·

Power window regulator switch assembly (for Rear Door RH, w/ Rear Power Window)


·

Located on the rear doors.

·

Each switch controls window operation for its respective window.


·

Front power window regulator motor assembly LH

·

Front power window regulator motor assembly RH

·

Rear power window regulator motor assembly LH (w/ Rear Power Window)

·

Rear power window regulator motor assembly RH (w/ Rear Power Window)


Receives switch signals and changes signals into motor activation. As a result, the window position changes.


SYSTEM FUNCTION


The power window control system has the following functions:

Function Outline
Manual up function Function that raises window while power window regulator switch assembly or power window regulator master switch assembly is pulled up (window stops as soon as power window regulator switch assembly or power window regulator master switch assembly is released).
Manual down function
·

for Driver Door:

Function that lowers window while power window regulator master switch assembly is pushed halfway down (window stops as soon as power window regulator master switch assembly is released).

·

except Driver Door:

Function that lowers window while power window regulator switch assembly is pushed down (window stops as soon as power window regulator switch assembly is released).


Auto down function (for Driver Door) Function that enables window to be fully opened by one full push of power window regulator master switch assembly.
Window lock function
·

Function where power window regulator switch assembly (for Front Passenger Door) and power window regulator switch assembly (for Rear Door, w/ Rear Power Window) operation is disabled when window lock switch of power window regulator master switch assembly is pressed.

·

Power window regulator switch assembly (for Front Passenger Door) and power window regulator switch assembly (for Rear Door, w/ Rear Power Window) can be operated when window lock switch is pressed again.


Remote control function Function that allows power window regulator master switch assembly to control manual up / down function of front passenger door power window and rear door power windows (w/ Rear Power Window).
