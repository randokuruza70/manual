LIGHTING (EXT)  REAR COMBINATION LIGHT ASSEMBLY(for Pickup)  INSTALLATION  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.

·

A bolt without a torque specification is shown in the standard bolt chart.



PROCEDURE

81563 1.INSTALL REAR NO. 1 COMBINATION LIGHT BRACKET LH (for Deckless)

a.

Install the rear No. 1 combination light bracket LH with the bolt.


81620D 2.INSTALL REAR COMBINATION LIGHT COVER LH (except Deckless)

a.

Install the rear combination light cover LH with the 4 bolts.


3.INSTALL SCREW GROMMET

a.

Attach the 2 claws to install a new screw grommet.

HINT:

Use the same procedure to install the screw grommet on the other side.



81560 4.INSTALL REAR COMBINATION LIGHT ASSEMBLY LH

a.

Connect the connector.


b.

Install the rear combination light assembly LH with the 4 screws.
