BRAKE (REAR)  REAR BRAKE(for Rear Disc Brake)  INSTALLATION  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

42431 1.INSTALL REAR DISC

a.

Align the matchmarks and install the rear disc.

HINT:

When replacing the disc with a new one, select the installation position where the disc has the smallest runout.



47775D 2.INSTALL REAR DISC BRAKE BUSH DUST BOOT

a.
0,0 0,0 false

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to the entire circumference of a new rear disc brake bush dust boot where it contacts the rear disc brake cylinder mounting LH, and the entire inner circumference of both ends.

HINT:

Apply at least 0.3 g (0.01 oz.) of lithium soap base glycol grease to the rear disc brake bush dust boot.


NOTICE:

Apply a sufficient amount of lithium soap base glycol grease to the entire circumference of the rear disc brake bush dust boot and rear disc brake cylinder mounting LH contact surfaces.



b.

Install the rear disc brake bush dust boot to the rear disc brake cylinder mounting LH.


47769A 3.INSTALL REAR DISC BRAKE CYLINDER SLIDE BUSH

a.
0,0 0,0 false

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to a new rear disc brake cylinder slide bush.

HINT:

Apply at least 0.8 g (0.03 oz.) of lithium soap base glycol grease to the rear disc brake cylinder slide bush.



b.

Install the rear disc brake cylinder slide bush to the rear disc brake cylinder mounting LH.


c.
0,0 0,0 false

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to the cylinder slide pin, insert it into the rear disc brake cylinder slide bush in order to properly grease the parts, and then remove it.


47713A 4.INSTALL REAR DISC BRAKE CYLINDER HOLE PLUG

a.
0,0 0,0 false

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to a new rear disc brake cylinder hole plug.


b.

Install the rear disc brake cylinder hole plug to the rear disc brake cylinder mounting LH.


47722A 5.INSTALL REAR DISC BRAKE CYLINDER MOUNTING LH

a.

Install the rear disc brake cylinder mounting LH and 2 plate washers with the 2 bolts.

Torque:
100 N*m (1020 kgf*cm, 74 ft.*lbf)

47771A 8.INSTALL REAR DISC BRAKE PAD WEAR INDICATOR PLATE

a.

Install the rear disc brake pad wear indicator plate to the inner side disc brake pad.

HINT:

Install the rear disc brake pad wear indicator facing downward.



04946 9.INSTALL REAR DISC BRAKE ANTI-SQUEAL SHIM

NOTICE:

If necessary, replace the anti-squeal shim when replacing the rear disc brake pad.


a.

Install the rear disc brake anti-squeal shims on each rear disc brake pad.

NOTICE:

There should be no oil or grease on the friction surfaces of the rear disc brake pads and the rear disc.



04466 10.INSTALL REAR DISC BRAKE PAD

a.

Install the 2 rear disc brake pads together with the rear disc brake anti-squeal shims to the rear disc brake cylinder mounting LH.


47750A 11.INSTALL REAR DISC BRAKE CYLINDER ASSEMBLY LH

a.
0,0 0,0 false

Lithium soap base glycol grease


Apply a light coat of lithium soap base glycol grease to the sliding surfaces of the 2 rear disc brake cylinder slide pins.


b.

Install the rear disc brake cylinder assembly LH with the 2 rear disc brake cylinder slide pins.

Torque:
88.3 N*m (900 kgf*cm, 65 ft.*lbf)


47787A 12.INSTALL DUST BOOT PROTECTOR LH

a.

Install the dust boot protector LH with the 2 bolts.

Torque:
7.85 N*m (80 kgf*cm, 69 in.*lbf)


47319F 13.CONNECT REAR FLEXIBLE HOSE LH

a.

Connect the rear flexible hose LH to the rear disc brake cylinder assembly LH with a new union bolt and a new gasket.

Torque:
30 N*m (306 kgf*cm, 22 ft.*lbf)
