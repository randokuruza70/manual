DOOR LOCK  WIRELESS DOOR LOCK CONTROL SYSTEM  No Answer-back  


DESCRIPTION

In some cases, wireless door lock control functions are normal but the hazard warning light answer-back function does not operate. In such cases, hazard warning light signal output from the door control with receiver ECU assembly may be malfunctioning.
