52553 9.INSTALL REAR BUMPER FILLER LH

HINT:

When installing the rear bumper filler LH, heat the vehicle body using a heat light.


Standard:
Item Temperature
Vehicle Body 40 to 60°C (104 to 140°F)


NOTICE:

Do not heat the vehicle body excessively.


a.

Clean the vehicle body surface.

i.

Using a heat light, heat the vehicle body surface.


ii.

Remove the double-sided tape from the vehicle body surface.


iii.

Wipe off any tape adhesive residue with cleaner.



b.

Install a new rear bumper filler LH.

i.

Remove the peeling paper from the rear bumper filler LH.

HINT:

After removing the peeling paper, keep the exposed adhesive free from foreign matter.



ii.

for Model Code #J76 Series:

1.

Install the rear bumper filler LH with 4 new grommets.

HINT:

Press the rear bumper filler LH firmly to install it.


2.

Install the 4 screws.



iii.

except Model Code #J76 Series:

1.

Install the rear bumper filler LH with 3 new grommets.

HINT:

Press the rear bumper filler LH firmly to install it.


2.

Install the 2 screws.

3.

Install the rear bumper arm mounting bracket LH with the 2 screws.

4.

Install the grommet.
