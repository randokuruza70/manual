THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  B2784  Antenna Coil Open / Short  


DESCRIPTION

The antenna coil receives key code signals from the transponder chip in the key grip. The coil is built into the transponder key amplifier, which amplifies the key code signals and outputs the signals to the transponder key ECU assembly.

This DTC is stored when there is an open or short in the antenna coil.

DTC No. Detection Item DTC Detection Condition Trouble Area Note
B2784 Antenna Coil Open / Short Antenna coil is open/shorted. Transponder key amplifier
