BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  ANTI-LOCK BRAKE SYSTEM  SYSTEM DESCRIPTION  


FUNCTION DESCRIPTION


a.

ABS (Anti-lock Brake System)

The ABS helps prevent the wheels from locking when the brakes are applied firmly or when braking on a slippery surface.



FUNCTION OF COMPONENTS


Component Function
Brake actuator assembly
·

Changes the fluid path based on the signals from the skid control ECU in order to control the fluid pressure that is applied to the wheel cylinders during the operation of the ABS.

·

Houses the skid control ECU.


Brake actuator assembly Skid control ECU
·

Processes the signals sent from each sensor to control the ABS.

·

Housed in the brake actuator assembly.


Skid control ECU ABS motor relay
·

Supplies power to the pump motor.

·

Housed in the skid control ECU.


ABS solenoid relay
·

Supplies power to each solenoid.

·

Housed in the skid control ECU.


Speed sensor Detects the wheel speed and sends the signal to the skid control ECU.
Deceleration sensor Detects the acceleration rate of the vehicle in the longitudinal direction and sends the signal to the skid control ECU.
Stop light switch Detects the brake operating conditions and inputs the results to the skid control ECU.
Combination meter assembly ABS warning light
·

Comes on to inform the driver that a malfunction in the ABS has occurred.

·

Blinks to output DTCs.
