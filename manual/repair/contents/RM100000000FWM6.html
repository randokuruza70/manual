DOOR LOCK  WIRELESS DOOR LOCK CONTROL SYSTEM  REGISTRATION  


PROCEDURE

1. DESCRIPTION OF CODE REGISTRATION

HINT:
·

Recognition code registration is necessary when replacing the door control transmitter or door control with receiver ECU assembly.

·

Add mode is used to register new recognition codes while still retaining codes already registered. This mode is used when a new door control transmitter is added. If the number of registered codes exceeds 4, the previously registered codes will be cleared in order, starting from the first registered code.

·

Rewrite mode is used to clear all the previously registered recognition codes in order to register new recognition codes. This mode is used when the door control transmitter or door control with receiver ECU assembly is replaced with a new one.

·

Confirmation mode is used to confirm how many recognition codes have already been registered before another recognition code is registered.

·

Clear mode is used to clear all the registered codes and disable the wireless door lock function. This mode is used when the door control transmitter is lost.

·

The registration procedures must be performed in order.



2. REGISTER RECOGNITION CODE

a.

Make sure the following conditions are met.

·

The ignition switch is off.

·

The driver side door is open (the other doors are closed).



b.

Perform the following operation:

i.

Turn the ignition switch to ON and then off (off → ON → off).

HINT:

The procedure should end with the ignition switch off.




c.

Perform the following operations within 40 seconds:

i.

Close and open the driver side door twice (Close → Open → Close → Open).


ii.

Turn the ignition switch to ON and then off (off → ON → off).

HINT:
·

The procedure should end with the driver side door open and the ignition switch off.

·

If even one of the following conditions is satisfied, registration mode is canceled.

·

These procedures are not completed within 40 seconds.

·

The ignition switch is turned to ON before the driver side door is operated.

·

The door control with receiver ECU assembly receives a valid RF signal during this procedure (door control transmitter switch is pressed).






d.

Perform the following operations within 40 seconds:

i.

Close and open the driver side door twice (Close → Open → Close → Open).


ii.

Close the driver side door and lock it.

HINT:
·

The procedure should end with the driver side door closed (locked) and the ignition switch off.

·

If even one of the following conditions is satisfied, registration mode is canceled.

·

These procedures are not completed within 40 seconds.

·

The ignition switch is turned to ON.

·

The door control with receiver ECU assembly receives a valid RF signal during this procedure (door control transmitter switch is pressed).






e.

Perform the following operations within 40 seconds:

i.

·

If even one of the following conditions is satisfied, registration mode is canceled.

·

These procedures are not completed within 40 seconds.

·

The number of ignition switch ON-off operations is 0, 4 or 6 or more.


·

In confirmation mode, a lock-unlock operation will occur once for each recognition code that has been registered. For example, if 2 recognition codes have been registered, the lock-unlock operation will occur 2 times.

·

In confirmation mode, if no recognition codes have been registered, the lock-unlock operation will occur 5 times.

·

If confirmation mode or clear mode is selected, the operation ends after the response to the selected mode completes.
