1HZ ENGINE CONTROL  ECD SYSTEM  41  Throttle Position Sensor Circuit  


DESCRIPTION

The throttle position sensor is mounted in the injection pump and detects the accelerator pedal opening angle. When the accelerator pedal is released, a voltage of approximately 2.6 to 3.8 V is applied to terminal VA of the emission control ECU. When the accelerator pedal is depressed, the voltage applied to terminal VA of the emission control ECU changes according to the position of the accelerator pedal and becomes approximately 0.1 to 1.3 V when the accelerator pedal is fully depressed.

HINT:

When the connector for the throttle position sensor is disconnected, diagnostic trouble code 41 is not output. Diagnostic trouble code 41 is output only when there is an open or short in the VA signal circuit of the throttle position sensor.
