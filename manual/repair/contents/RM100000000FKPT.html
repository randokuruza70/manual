BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  ANTI-LOCK BRAKE SYSTEM  PRECAUTION  


TROUBLESHOOTING PRECAUTION


a.

When there are problems with terminal contact points or part installation problems, removal and installation of the suspected problem parts may completely or temporarily return the system to normal.


b.

In order to determine the malfunctioning area, check the conditions at the time the malfunction occurred, such as stored DTC output and the freeze frame data, and record them before disconnecting each connector or removing and installing parts.


c.

Since the system may be influenced by malfunctions in other systems, be sure to check for DTCs in the other systems.



HANDLING PRECAUTION


a.

Be sure to remove and install the ECU, brake actuator and each sensor, etc. with the ignition switch off unless otherwise specified in the inspection procedure.


b.

If the ECU, the brake actuator or a sensor has been removed and installed, it is necessary to check the system for problems after the parts have been reassembled. Check for DTCs using the GTS, and also check that the system functions and signals received by the ECU are normal using Test Mode.



DTC PRECAUTION


a.

Some DTCs cannot be cleared only by repairing the malfunctioning parts. If the DTC is output after repair work, turn the ignition switch off to clear the DTC.

NOTICE:

If a part is still malfunctioning even after the DTC is cleared, the DTC is stored again.
