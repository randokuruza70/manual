RM25T0E _59 Vehicle Exterior _029935 WINDOW / GLASS _0137674 BACK WINDOW GLASS A

WINDOW / GLASS  BACK WINDOW GLASS  DISASSEMBLY  


PROCEDURE

64851 1.REMOVE BACK WINDOW SLIDE GLASS STOP (for Slide Type)

a.

Remove the screw and back window slide glass stop.


64885 2.REMOVE BACK WINDOW VERTICAL FRAME (for Slide Type)

a.

Remove the 4 screws and 2 back window vertical frames.


b.

Remove the 2 center back window glass weatherstrips and 2 back window channel weatherstrips from the 2 back window vertical frames.


64822A 3.REMOVE BACK WINDOW SLIDE GLASS LH (for Slide Type)

a.

Pull apart the back window frame, remove the back window slide glass LH as shown in the illustration.


b.

Remove the center back window channel from the back window glass channel.


c.

Remove the back window glass channel from the back window slide glass LH.


d.

Remove the screw, the back window lock catch, back window lock retainer and back window lock packing.


64821A 4.REMOVE BACK WINDOW SLIDE GLASS RH (for Slide Type)

a.

Pull apart the back window frame, remove the back window slide glass RH as shown in the illustration.


b.

Remove the back window glass channel from the back window slide glass RH.


c.

Remove the screw, the back window lock assembly, back window lock retaner and back window lock packing.


64814 5.REMOVE BACK WINDOW CORNER GLASS LH (for Slide Type)

a.

Pull apart the back window frame, remove the back window corner glass LH as shown in the illustration.


b.

Remove the back window glass channel from the back window corner glass LH.


64813 6.REMOVE BACK WINDOW CORNER GLASS RH (for Slide Type)

HINT:

Use the same procedure described for the back window corner glass LH.


64812C 7.REMOVE BACK WINDOW GLASS WEATHERSTRIP (for Slide Type)

a.

Remove the back window glass weatherstrip.


64825 8.REMOVE UPPER BACK WINDOW CHANNEL RUN (for Slide Type)

a.

Remove the upper back window channel run from the back window frame.


64826 9.REMOVE LOWER BACK WINDOW CHANNEL RUN (for Slide Type)

a.

Remove the lower back window channel run from the back window frame.


64829 10.REMOVE BACK WINDOW FRAME FILM (for Slide Type)

a.

Remove the 6 back window frame films from the back window frame.


64812C 11.REMOVE BACK WINDOW GLASS WEATHERSTRIP (for Fixed Type)

a.

Remove the back window glass weatherstrip from the back window glass.
