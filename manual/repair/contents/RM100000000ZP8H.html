CRUISE CONTROL  CRUISE CONTROL SYSTEM  FREEZE FRAME DATA  


CHECK FREEZE FRAME DATA (READ BY GTS)


HINT:

The ECU data indicating the vehicle status is stored in the ECU as freeze frame data simultaneously as the DTC is stored.


a.

Connect the GTS to the DLC3.


b.

Turn the ignition switch to ON.


c.

Turn the GTS on.


d.

Enter the following menus: Powertrain / Cruise Control / Trouble Codes.

Powertrain > Cruise Control > Trouble Codes
50001 237

e.

From the DTC screen, select the DTC (snow icon displayed) to which freeze frame data is recorded.
