THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  B2797  Communication Malfunction No. 1  


DESCRIPTION

This DTC is stored when a communication error occurs between the transponder key amplifier and transponder key ECU assembly. Some possible reasons for the communication error are: 1) 2 or more ignition keys are positioned too close together, or 2) noise is occurring in the communication line.
