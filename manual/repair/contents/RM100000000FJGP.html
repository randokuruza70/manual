1GR-FE ENGINE CONTROL  SFI SYSTEM  DTC CHECK / CLEAR  


NOTICE:

When the diagnosis system is changed from the normal mode to check mode or vice versa, all DTCs and freeze frame data stored in normal mode are cleared. Before changing modes, always check and make a note of DTCs and freeze frame data.


HINT:
·

DTCs which are stored in the ECM can be output to the GTS. The GTS can display the current and pending DTCs.

·

Some DTCs are not stored if the ECM does not detect the same malfunction again during a second consecutive driving cycle. However, such malfunctions, detected on only one occasion, are stored as pending DTCs.



CHECK DTC


a.

Connect the GTS to the DLC3.


b.

Turn the ignition switch to ON.


c.

Turn the GTS on.


d.

Enter the following menus: Powertrain / Engine and ECT / Trouble Codes.


e.

Check the DTC(s) and freeze frame data, and then write them down.


f.

Check the details of the DTC(s) (See page Engine / Hybrid System>1GR-FE ENGINE CONTROL>SFI SYSTEM>DIAGNOSTIC TROUBLE CODE CHART201408,201608,_51,_029758,_0135343,RM100000000FJFO,201608,999999,_51,_029758,_0135343,RM1000000011O8H,).



CLEAR DTC


a.

Connect the GTS to the DLC3.


b.

Turn the ignition switch to ON.


c.

Turn the GTS on.


d.

Enter the following menus: Powertrain / Engine and ECT / Trouble Codes.


e.

Clear the DTCs.
