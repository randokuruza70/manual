THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  B279A  Theft Deterrent System Communication Line High Fixation  


DESCRIPTION

If the communication line (EFIO-IMI) to the transponder key ECU assembly is stuck high (e.g. shorted to +B), the ECM stores this DTC.

DTC No. Detection Item DTC Detection Condition Trouble Area Note
B279A Theft Deterrent System Communication Line High Fixation Communication line (EFIO-IMI) between the ECM and transponder key ECU assembly is stuck high.
·

Harness or connector

·

ECM

·

Transponder key ECU assembly


DTC Output Confirmation Operation:

Turn the ignition switch to ON and wait 6 seconds.
