LIGHTING (EXT)  REAR COMBINATION LIGHT ASSEMBLY(for Canvas Top)  INSTALLATION  


CAUTION / NOTICE / HINT

HINT:
·

Use the same procedure for the RH and LH sides.

·

The procedure listed below is for the LH side.



PROCEDURE

1.INSTALL SCREW GROMMET (for Rear Body Side)

a.

Attach the 2 claws to install a new screw grommet.

HINT:

Use the same procedure to install the screw grommet on the other side.



2.INSTALL REAR COMBINATION LIGHT ASSEMBLY LH (for Rear Body Side)

a.

Connect the connector.


b.

Pass the wire harness into the vehicle and connect the grommet.


c.

Install the rear combination light assembly LH with the 3 screws.

b.

Connect the connector.
