1HZ FUEL  FUEL FILTER  REPLACEMENT  


CAUTION / NOTICE / HINT

HINT:

When replacing the filter, use TOYOTA genuine parts or equivalent parts.


PROCEDURE

1.DRAIN FUEL FROM FUEL FILTER

a.

*a Container


Connect a hose to the drain cock. Place the other side of the same hose into a container.


b.

Loosen the drain cock to drain fuel.


c.

Tighten the drain cock.


23390 2.REMOVE FUEL FILTER ELEMENT ASSEMBLY

a.

w/ Fuel Heater:

Disconnect the fuel heater connector.


b.

Slide the 2 clamps and disconnect the 2 fuel hoses from the fuel filter cap assembly.


c.

Using a 5 mm hexagon socket wrench, remove the 3 bolts and fuel filter cap assembly.


d.

Remove the O-ring and retainer.


e.

Remove the fuel filter element assembly from the fuel filter case.

NOTICE:

Do not damage the fuel filter element assembly and fuel filter case.


HINT:

Tape the screwdriver tip before use.



f.

Disconnect the fuel sediment switch connector.


g.

Detach the clamp and remove the fuel filter case.


23390 3.INSTALL FUEL FILTER ELEMENT ASSEMBLY

a.

Connect the fuel sediment switch connector.


b.

Attach the clamp and install the fuel filter case.


c.
0,0 0,0 false

No Foreign Matter


Install the fuel filter element assembly to the fuel filter case.

NOTICE:

Make sure that there is no foreign matter on the areas of the fuel filter case shown in the illustration.



d.

Install the retainer and a new O-ring.

NOTICE:

Make sure to install the O-ring with the rounded side facing upward and the retainer as shown in the illustration.



e.

Using a 5 mm hexagon socket wrench, install the fuel filter cap assembly with the 3 bolts.

Torque:
4.5 N*m (46 kgf*cm, 40 in.*lbf)

NOTICE:
0,0 0,0 false

No Foreign Matter


·

Make sure that there is no foreign matter on the areas of the fuel filter cap assembly shown in the illustration.

·

Do not turn the fuel filter cap as the O-ring will be damaged.



HINT:

Further tighten the first bolt tightened above so that its torque value is slightly more than the torque specification.



f.

Connect the 2 fuel hoses to the fuel filter cap assembly, and slide the 2 clamps to secure the hoses.


g.

w/ Fuel Heater:

Connect the fuel heater connector.


4.BLEED AIR FROM FUEL FILTER

a.

Using the hand pump, bleed air from the fuel system until pumping becomes difficult.


5.INSPECT FOR FUEL LEAK

a.

Start the engine. Check that leaking sounds, sucking sounds or other unusual sounds cannot be heard.
