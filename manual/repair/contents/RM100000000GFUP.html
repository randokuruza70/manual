THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  B2796  No Communication in Immobiliser System  B2798  Communication Malfunction No. 2  


DESCRIPTION

DTC B2796 is stored when a key is inserted into the ignition key cylinder but no communication occurs between the key and transponder key ECU assembly.

DTC B2798 is stored when a key is inserted into the ignition key cylinder but a communication error occurs between the key and transponder key ECU assembly.

DTC No. Detection Item DTC Detection Condition Trouble Area Note
