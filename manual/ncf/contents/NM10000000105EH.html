BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  BRAKE CONTROL SYSTEM  CONTROL  EMERGENCY BRAKE SIGNAL  


SYSTEM CONTROL


a.

The emergency brake signal system is a system which helps reduce the danger of the rear-end collision by automatically flashing each turn signal light to warn vehicles at the rear during emergency braking.



b.

The skid control ECU requests an emergency brake signal to the combination meter assembly when detecting sudden braking according to each sensor and the operation status of the brake pedal. As a result, the combination meter assembly flashes each turn signal light.


c.

The emergency brake system does not operate while the turn signal light is flashing due to the hazard warning signal switch operation.
