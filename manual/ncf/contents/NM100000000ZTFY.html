BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  BRAKE CONTROL SYSTEM  DETAILS  STEERING ANGLE SENSOR  


CONSTRUCTION


a.

The steering angle sensor detects the steering direction and angle, and sends this signal to the skid control ECU.


b.

The steering angle sensor contains 2 sets of magnetic reluctance elements that detect the rotational movement of a magnet that is built into the detection gear. Thus, the sensor detects the changes that occur in the magnetic reluctance elements along with the rotational movement of the detection gear, in order to detect the rotational movement of the steering wheel.
