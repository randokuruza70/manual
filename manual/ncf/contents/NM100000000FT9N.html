FUNCTION


a.

The power door lock control system has the following functions:

Function Outline
Lock and Unlock This function can lock or unlock all doors by operation of the door control switch.
Key-linked Lock When the ignition key is inserted into the driver door key cylinder and turned, all doors are locked.
Key-linked Unlock When the ignition key is inserted into the driver door key cylinder and turned, all doors are unlocked.
