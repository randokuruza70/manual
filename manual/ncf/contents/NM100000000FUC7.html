METER / GAUGE / DISPLAY  CLOCK SYSTEM  CONTROL  CLOCK CONTROL  


FUNCTION


a.

Clock Function

i.

The time can be displayed.


ii.

The time is set via the H, M and SET switches in the clock assembly.



b.

Alarm Function (Multifunction Digital Clock Only)

i.

An alarm can be set to sound at a certain time.


ii.

The alarm only sounds when the ignition switch is turned to ACC or off.



c.

Outside Temperature Display Function (Multifunction Digital Clock Only)

i.

The ambient temperature is displayed.


ii.

The ambient temperature display range is from -30°C to 50°C.



d.

Stopwatch Function (Multifunction Digital Clock Only)

i.

The stopwatch function counts time.


ii.

This function can count up to 19:59.9. When the counted time exceeds this value, the display is reset to 00:00.0 and counting continues.


iii.

A lap mode, which can stop the count display when the LAP switch is pressed during a count, is used. However, the count continues and when the LAP switch is pressed again, the continuing count is displayed.


iv.

The count is reset by first stopping the count, and then pressing the RESET switch.
