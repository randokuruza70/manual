OUTLINE


a.

Front Differential

i.

An RD20A type front differential is used.


ii.

A differential lock system is provided depending on the model.

RD20A Type Front Differential

RD20A Type Front Differential (with Differential Lock Mechanism)



b.

Rear Differential

i.

A BD24A type rear differential is used.


ii.

A multi-disc type Limited Slip Differential (LSD) is provided depending on the model.


iii.

A differential lock system is provided depending on the model.

BD24A Type Rear Differential

BD24A Type Rear Differential (with LSD Mechanism)

BD24A Type Rear Differential (with Differential Lock Mechanism)
