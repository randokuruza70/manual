METER / GAUGE / DISPLAY  METER / GAUGE SYSTEM  GENERAL  GENERAL  


OUTLINE


a.

2 types of analog display type combination meter assemblies are used.


b.

A meter ECU and a buzzer are enclosed in the combination meter assembly.


c.

A front seat belt warning function is provided on models for G.C.C. countries. If the driver seat belt is not buckled, this system blinks the seat belt warning light and sounds the buzzer as a reminder.


d.

A front seat belt warning function is provided on models for Australia. If the driver or front passenger* seat belt is not buckled, this system illuminates the seat belt warning light and sounds the buzzer as a reminder.

*: Single cab pick-up



e.

A rheostat is used to make changing the brightness of the combination meter possible. The brightness level is changed on the Liquid Crystal Display (LCD) for ODO/TRIP display.
