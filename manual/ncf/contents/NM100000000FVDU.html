1GR-FE FUEL  FUEL SYSTEM  CONTROL  SUB FUEL TANK SYSTEM  


FUNCTION


a.

When the driver presses the sub-tank select switch, the fuel pump and fuel tank solenoid return valve are switched to select the main fuel tank or sub fuel tank to use.


b.

Because a fuel inlet is provided separately for the main fuel tank and sub fuel tank, the main fuel tank and sub fuel tank should be distinguished when refueling.
