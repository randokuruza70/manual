WINDOW / GLASS  POWER WINDOW CONTROL SYSTEM  CONTROL  POWER WINDOW CONTROL  


SYSTEM CONTROL


a.

The power window control system has the following functions:

Function Outline
Manual Up-and-down When the ignition switch is turned to ON and the power window regulator master switch or power window regulator switch of the driver seat, passenger seat or rear seat* is pulled up or lightly pressed down, the manual up-and-down function enables an open or close operation for the appropriate window.
One-touch Auto Down (for Driver Door) The one-touch auto down function enables the window to be fully opened by firmly operating the power window regulator master switch assembly.
Window Lock
·

Function where front passenger side power window regulator switch and rear power window regulator switch* operation is disabled when window lock switch of master switch is pressed.

·

Front passenger side power window regulator switch and rear power window regulator switch* can be operated when window lock switch is pressed again.



*: Models with rear power window system
