FUNCTION OF MAIN COMPONENTS


Component Function

*1: Models with rear door

*2: Models with back door

Door Control Transmitter
·

Contains the lock and unlock buttons.

·

Transmits weak electric waves (recognition codes and function codes) to the door control receiver.


Door Control Receiver Receives weak electric waves (recognition codes and function codes) from the door control transmitter and changes the waves to code data.
Front Door Courtesy Light Switch Assembly LH and RH
·

Detects the door status (open or closed) and outputs data to the door control receiver via the combination meter assembly.

·

Turns on when the door is opened and turns off when the door is closed.


Rear Door Courtesy Light Switch Assembly LH and RH*1
Back Door Courtesy Light Switch Assembly*2
Front Door Lock Assembly (Driver Side) Operates the motor in response to signals from the door control receiver and locks and unlocks the doors.
Front Door Lock Assembly (Passenger Side)
Rear Door Lock Assembly LH and RH*1
Back Door Lock Assembly*2
Combination Meter Assembly Receives the signal from the door control transmitter and controls the hazard warning lights as answer back.


FUNCTION


a.

The wireless door lock control system has the following functions:

Function Outline
All Doors Lock Pressing the lock button of the door control transmitter locks all doors and the back door.
All Doors Unlock Pressing the unlock button of the door control transmitter unlocks all doors and the back door.
Answer Back When the doors are being locked or unlocked using the door control transmitter, the hazard warning lights blink once while locking and twice while unlocking.
