WINDOW / GLASS  WINDSHIELD / WINDOW GLASS SYSTEM  CONTROL  REAR DEFOGGER CONTROL  


FUNCTION OF MAIN COMPONENTS


Component Function
Rear Window Defogger Switch Assembly Operates the rear window defogger system.
Defogger Wire Applies electrical current to a rear defogger wire in the back door glass to generate heat and remove ice from the swing back door glass.


SYSTEM COTNROL


a.

The rear window defogger system is activated when the ignition switch is turned to on and the rear window defogger switch assembly is pushed.
