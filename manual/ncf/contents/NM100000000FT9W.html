THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  GENERAL  GENERAL  


OUTLINE


a.

The immobiliser system compares the ID code registered in the transponder key amplifier with the ID code of the transponder chip embedded in the ignition key. The system unsets if these ID codes match. Thus, the transponder key amplifier and the ECM communicate with each other to authorize fuel injection and ignition*, enabling the engine to start.

HINT:

*: Models with 1GR-FE engine

When replacing the transponder key amplifier or making a new ignition key, the key recognition code must be registered. For details, refer to the Service Bulletin.
