WINCH  WINCH SYSTEM  CONTROL  WINCH CONTROL  


FUNCTION OF MAIN COMPONENTS


Component Function
Winch Assembly Drives the built-in motor assembly according to signals from the winch control switch assembly.
Winch Control Switch Assembly Power Indicator Light Illuminates in green when the winch is in an operable condition.
Warning Indicator Light Illuminates in amber when the temperature of the motor assembly exceeds the specified value.
Buzzer Sounds at the same time the warning indicator light illuminates.
Operation Switch
·

Switch operation signals are output to the winch assembly.

·

The winch is operated by continuously pressing the switch to the IN or OUT side. If released, the switch automatically moves to the OFF position.




FUNCTION


Function Outline
Wind Up
·

If the operation switch of the winch control switch assembly is moved to the IN side, the wire rope is pulled in.

·

If the operation switch is released, the wind up function automatically stops.

·

When performing the wind up operation, put the clutch in the lock condition.


Wind Out (with Motive Power)
·

This function is for the wind out operation using the winch control switch assembly.

·

If the operation switch of the winch control switch assembly is moved to the OUT side, the wire rope can be pulled out.

·

If the operation switch is released, the wind out function automatically stops.

·

When performing the wind out operation, put the clutch in the locked condition.


Wind Out (without Motive Power)
·

This function is for the wind out operation without using the winch control switch assembly.

·

When performing the wind out operation, use the clutch in the free condition.


Motor Overheating Stop If the motor assembly temperature exceeds a specified value, the warning indicator light illuminates, the buzzer sounds and the winch assembly winding operation stops. If the motor assembly temperature decreases to the specified value or less, the warning indicator light illumination and buzzer sounding end, and operation of the winch becomes possible.
