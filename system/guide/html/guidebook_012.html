New Car Features

Overview

New Car Features consists of the following Categories, each of which contains related Sections.Each Section consists of Overview, Control, and Details, which clarify the details of the Section. (Some items may not be displayed in some Sections.)

Select the item to display from the tree and display the Contents.

Displaying PDF data

Vehicle orthogonal views, specifications, etc., can be displayed as PDF data in separate windows by clicking on “PDF”.

Enlarged video display

Click on “Enlarge” within a screen displaying video to enlarge that video and display it in a separate window.
